# CHAPTER 0　このゲームについて

## 0-1　概要

舞台は近未来の架空日本。魔法はインフラとして社会に組み込まれ、その裏側では噂が質量を持ち、現実にバグとして侵食する怪異が日常的に発生している。

あなたは討伐者だ。怪異を調査し、核とルールを解明し、討伐する。

このゲームの大きな特徴は三つだ。

**① 《共鳴記録》システム**
能力値のランク（D〜S）に応じてダイスの数が変わる。判定のたびに、その行為に紐づいた感情の波が討伐者の体内で共鳴し、**感情共鳴メーター**に積み重なっていく。共鳴が深まればギフトが解放されるが、特定の感情が臨界まで達すると代償が訪れる。

**② 怪異予兆カード**
GMはセッション開始前に各PCへ「怪異予兆カード」を1枚配る。カードに書かれた条件を満たすと公開され、物語に新たな次元が加わる。

**③ 二重のリスク**
魔法は怪異を生む。異能は自分を変質させる。侵食率（身体的変容）と感情共鳴メーター（心理的変容）の二軸が、討伐者が「何かを失っていく」過程を記録する。

## 0-2　面白さの三層構造

| 層 | 体験の核心 |
|---|---|
| **パズルとしての討伐** | 怪異のルールと核を解明して初めて倒せる。知識が命を救う。 |
| **感情のジレンマとしての判定** | 強く行動するほど感情が共鳴する。共鳴は力だが、偏ると代償が来る。 |
| **問いとしての世界観** | 怪異と神の違いは恐れか信仰か。討伐者たちは正しいことをしているのか。 |

# CHAPTER 1　世界観

## 1-1　怪異のメカニズム

怪異とは、集合的な噂・言説・信念が臨界点を超えた時に現実へと侵食するバグだ。全ての怪異は**核（コア）**と**ルール**を持つ。

| 概念 | 内容 |
|---|---|
| **核（コア）** | 怪異の存在の中心。物・場所・記録媒体に宿る。稀に人間に宿る。破壊で怪異は消滅する。 |
| **ルール** | 怪異が人を害する条件と行動原則。ルールを破るほど捕捉される。 |

## 1-2　能力体系

| 種別 | 概要 | リスク |
|---|---|---|
| **魔法（チートコード）** | 世界のソースコードへの直接介入。訓練で習得。インフラとして資格制度がある。 | 使用規模に比例して怪異を生む |
| **異能（アプリケーション）** | 怪異と同じ発生原理の延長線上にある力。体験・執着・信念から発現。 | 使用のたびに侵食率が蓄積する |

## 1-3　三組織

| 組織 | 概要 |
|---|---|
| **祓部（はらえべ）** | 魔導省内部の公的討伐機関。神職系称号を持つ専門家集団。データベース・情報優位。 |
| **傭兵集団** | 独立系企業バックの民間討伐力。装備性能・戦闘技術優位。二つ名持ちが伝説扱いされる。 |
| **無所属** | 国にも企業にも属さない個人または小集団。法的保護なし。曰く付き装備・独自情報網。 |

# CHAPTER 2　基本判定システム

## 2-1　能力値とランク

キャラクターは7つの能力値を持つ。各能力値はランク（D〜S）で表される。

| 能力値 | 読み | 主な用途 |
|---|---|---|
| **体** | たい | 格闘・突破・耐久 |
| **疾** | はや | 先手・回避・追跡（イニシアチブに使用） |
| **識** | しき | 調査・知識・文献・解明 |
| **判** | はん | 解明宣言・看破・戦術 |
| **視野** | しや | 怪異感知・観察・証言聴取 |
| **術** | じゅつ | 魔法行使・魔導具操作・装備の高度な運用 |
| **魂** | こん | 異能制御・侵食抵抗・信念維持 |

| ランク | ダイス数 | 特徴 |
|---|---|---|
| **D** | 1d6 | 共鳴なし |
| **C** | 2d6 | 余り1個→共鳴へ |
| **B** | 3d6 | 余り2個→共鳴へ |
| **A** | 4d6 | 余り3個→共鳴へ |
| **S** | 4d6+固有特典 | 余り3個→共鳴へ＋追加効果 |

## 2-2　判定手順

1. GMが要求する能力値を宣言
2. ランクに応じたd6を振る
3. **1個を「達成値ダイス」として選択**
4. **残りの中から1個を「共鳴ダイス」として選択**（残り0個=選択不要）
5. 達成値（出目＋修正）が**4以上で成功**

| 出目 | 効果 |
|---|---|
| **6** | **スペシャル**：自動成功＋追加効果 |
| 4〜5 | 成功 |
| 2〜3 | 失敗 |
| **1** | **ファンブル**：自動失敗＋ファンブル表を参照 |

<a id="ch3"></a>
# CHAPTER 3　《共鳴記録》システム

## 3-1　設計思想

怪異と戦い続ける討伐者の体は、少しずつ怪異の波長に共鳴していく。それは能力として現れることもあるし、人間としての何かを削ることもある。**《共鳴記録》はその蓄積を記録するシステムだ。**

## 3-2　6種の感情共鳴メーター

各メーターの最大値は**10点**。

| メーター | 色 | 上昇する判定の文脈 | 臨界の代償 |
|---|---|---|---|
| **恐怖（FEAR）** | 暗赤 | 逃走・退避・防御に失敗した時、怪異のルールを破った時 | 次の【魂】判定が自動ファンブル。侵食率+10% |
| **怒り（RAGE）** | 赤橙 | 攻撃判定スペシャル時、仲間が傷ついた時の行動 | 1ラウンド行動が強制：最も近い敵への最大火力攻撃のみ |
| **哀愁（SORROW）** | 青 | 解明鍵を1つ入手した時、NPCとの別れの場面 | 次の調査判定がすべて−1。信念ポイント1消費 |
| **焦燥（HASTE）** | 橙 | 制限ラウンド残り1で行動した時、手がかりを失った時 | 次の行動宣言を相手より先に強制公開（奇襲不可） |
| **渇望（THIRST）** | 紫 | 異能・怪異核素材装備を使用した時、侵食ロール後 | 侵食率+5%。次の侵食ロールの失敗閾値が1段階悪化 |
| **浄化（PURGE）** | 白緑 | 解明完了宣言成功時、信念ポイントを全消費した時 | 代償なし（臨界で大浄化ギフト発動→リセット） |

## 3-3　共鳴メーターの上昇タイミング

共鳴ダイスを選択した後、**その判定の文脈**を参照して上昇するメーターを決める。

- **共鳴ダイスの出目は関係しない**（出目はGMへの参考情報として見せるだけ）
- 文脈が複数のメーターに該当する場合、PLが1つ選ぶ
- ランクDの判定（ダイス1個）は共鳴なし

| 状況 | 上昇量 |
|---|---|
| 通常の判定（成功・失敗問わず） | 対応メーター+1 |
| スペシャル（出目6）の判定 | 対応メーター+2 |
| ファンブル（出目1）の判定 | 恐怖メーター+2（強制） |

## 3-4　覚醒段階と臨界

| 蓄積点 | 覚醒段階 | 解放されるギフト |
|---|---|---|
| 1〜3 | **第一段階** | 初級覚醒ギフト |
| 4〜6 | **第二段階** | 中級覚醒ギフト |
| 7〜9 | **第三段階** | 上級覚醒ギフト |
| 10 | **臨界（CRITICAL）** | 代償発動（浄化のみ大浄化恩恵） |

臨界後、対応するメーターは**0にリセット**。浄化のみ恩恵後リセット。全員のメーターは卓全体に公開される。

<a id="ch4"></a>
# CHAPTER 4　怪異との戦闘《核護衛戦》

## 4-1　設計思想

怪異の核は剥き出しでない。核には**護衛（ガーディアン）**が取り巻いている。護衛を突破せずに核を攻撃することは、通常不可能だ。

**討伐の本質は「護衛突破＋核へのダメージ」を制限ラウンド内に完了させること。**

## 4-2　核護衛戦の基本構造

```
【核（コア）】
  ↑  核を守る
【護衛1】 【護衛2】 【護衛3】
  ↕  PCたちが対峙する
【討伐者チーム】
```

### 制限ラウンド（タイムリミット）

| 怪異等級 | 制限ラウンド | 護衛の数 |
|---|---|---|
| 五級 | 3ラウンド | なし（核直接） |
| 四級 | 3ラウンド | 護衛1体 |
| 三級 | 3ラウンド（+1回延長可） | 護衛2体 |
| 二級 | 4ラウンド | 護衛2〜3体 |
| 一級 | 4ラウンド（+1回延長可） | 護衛3体 |
| 特級 | 封印クロック方式 | 護衛多数＋特殊ギミック |

解明完了宣言が成立している場合、「制限ラウンド+1」の延長を**1回だけ**適用できる。

## 4-3　護衛（ガーディアン）

### 護衛データの構成

| 項目 | 内容 |
|---|---|
| **耐久値（HP）** | 護衛の体力。0で機能停止。 |
| **行動値** | 護衛の攻撃・妨害の強度 |
| **特性** | 護衛固有のルール |
| **連鎖条件** | 1ラウンドで耐久値以上のダメージを与えた場合の特殊効果 |

### 連鎖ダメージ

護衛に対して**1ラウンド内で耐久値以上のダメージ**を与えると：
1. 護衛は**そのラウンド終了まで行動不能**
2. 護衛の耐久値と**同点のダメージ**が核に自動伝播
3. スペシャル（出目6）による攻撃なら伝播ダメージが**×1.5（端数切り上げ）**

> **例：** 耐久値5の護衛に7ダメージ→護衛行動不能＋核に5ダメージ自動伝播。スペシャルなら核に8ダメージ。

### 護衛特性の例

| 特性名 | 効果 |
|---|---|
| **連結（CHAIN）** | この護衛が行動不能になると隣の護衛のHP+3回復 |
| **干渉（BLOCK）** | この護衛がいる限り、核への直接攻撃は−3修正 |
| **反射（REFLECT）** | ダメージの半分（切り捨て）が攻撃者に跳ね返る |
| **再生（REGEN）** | ラウンド開始時にHP+2回復 |
| **感染（INFECT）** | この護衛からダメージを受けると恐怖メーター+1 |

## 4-4　行動順（イニシアチブ）

1. GMが「行動宣言フェイズ」を開始
2. PLは全員同時に行動を宣言
3. **疾ランクが高いPC→低いPC→護衛→核**の順で解決
4. 同ランクの場合、PLが望む順番で解決

### 行動の種類

| 行動 | 内容 | 使用能力値 |
|---|---|---|
| **攻撃（ATTACK）** | 護衛または核へのダメージ行動 | 体/疾（武器による） |
| **魔法攻撃（SPELL）** | 魔法を使った攻撃または支援 | 術 |
| **援護（COVER）** | 別のPCへの攻撃を肩代わりする | 体 |
| **解明（DECODE）** | 戦闘中に解明鍵を進める | 識/視野 |
| **干渉（DISTURB）** | 護衛の特性を一時的に無効化 | 判 |
| **回避専念（EVADE）** | 攻撃不可だがダメージ無効 | 疾 |
| **浄化（CLEANSE）** | 魂で判定。成功で共鳴メーター1種を−2点 | 魂 |
| **異能発動（ABILITY）** | 異能を使用（侵食ロール発生） | 魂 |

## 4-5　ダメージ計算

```
ダメージ = 達成値 + 武器修正 − 対象の防御値
```

| 達成値 | 武器なし | 武器あり |
|---|---|---|
| 4 | 1 | 1+武器修正 |
| 5 | 2 | 2+武器修正 |
| 6（スペシャル） | 3+特典 | 3+武器修正+特典 |

### 核への直接攻撃

解明完了宣言を経た後のみ**核への直接攻撃が可能**。護衛が全て行動不能または破壊されている場合も可能。

核への直接攻撃が成功すると討伐クロックが進行する。

## 4-6　護衛の行動

| 行動 | 効果 |
|---|---|
| **攻撃** | 任意のPCに行動値分のダメージ |
| **妨害** | 任意のPCの次の判定に−1修正 |
| **庇い** | 核への攻撃を自分で受ける（1ラウンドに1回） |
| **強化** | 別の護衛のHPを3回復 |

## 4-7　討伐クロックと終了条件

| 怪異等級 | 初期クロック値 |
|---|---|
| 五級 | 4 |
| 四級 | 6 |
| 三級 | 8 |
| 二級 | 10 |
| 一級 | 12 |
| 特級 | 封印クロック（別途設定） |

| 行動 | クロック進行 |
|---|---|
| 核への攻撃（成功） | −1 |
| 核への攻撃（スペシャル） | −2 |
| 護衛への連鎖ダメージ伝播 | −（伝播ダメージ分） |
| 解明完了後の浄化ギフト発動 | −2〜−4（GMが設定） |
| 核への直接攻撃（成功・解明完了後） | 残クロック全消化（即時討伐） |

**クロックが0になると討伐成功。**

特級怪異は討伐クロックの代わりに**封印クロック**を用いる。クロック最大値に達すると「管理封印状態」でシナリオクリア。

## 4-8　共鳴記録と戦闘

| 戦闘状況 | 上昇するメーター |
|---|---|
| 攻撃判定（命中） | 怒り +1 |
| 攻撃判定（スペシャル命中） | 怒り +2 |
| ダメージを受けた時 | 恐怖 +1 |
| 護衛への連鎖ダメージ成功 | 怒り +2 |
| 制限ラウンド残り1 | 焦燥 +1（全PC強制） |
| 仲間が行動不能になった場面 | 哀愁 +1（任意）or 怒り +1（任意） |
| 魔法攻撃を使用した場面 | 渇望 +1 |
| 異能を使用した場面 | 渇望 +1 |

# CHAPTER 5　調査・解明・討伐の三段階

## 5-1　概要

```
[第一段階] 調査プロセス
    ↓ 四つの解明鍵の収集
[第二段階] 解明プロセス
    ↓ 解明完了宣言
[第三段階] 討伐プロセス《核護衛戦》
    ↓ 討伐クロック消化
[終了] 討伐完了・撤退・封印
```

戦闘はあくまで出口だ。最初の二段階が生死を分ける。

## 5-2　四つの解明鍵

| 鍵 | 内容 | 主な判定 |
|---|---|---|
| **①怪異の正体** | 古い/新しい怪異か、元になった噂の内容、発生時期 | 識・視野 |
| **②核の所在** | 核が宿っている物・場所・人間 | 視野・識 |
| **③被害パターン** | ルールの構造、誰が狙われるか、条件 | 判・識 |
| **④ルールの全容** | 怪異のルールと弱点・例外条件 | 判 |

### 改造個体・別個体の罠

文献との齟齬、被害パターンの不一致、核の所在の混乱——これらは「何かがおかしい」サインだ。

- **改造個体**（ルール変質50%未満）：解明中にある一点でルールが決定的に食い違う。
- **別個体**（ルール変質50%以上）：解明を進める途中で全く別の核とルールを持つと判明する。それまでの解明が全て無効になる。

文献調査（識）での解明鍵①には−1修正（改造個体の場合）。無所属は「改造個体センサー」でヒントを1回GMに要求できる。

## 5-3　解明完了宣言

| 達成鍵数 | 効果 |
|---|---|
| **4鍵全て** | 討伐クロック−1先行・浄化メーター+1・戦闘中の制限ラウンド+1延長権 |
| **3鍵** | ±0（通常戦闘開始） |
| **2鍵以下で強行** | 全判定に−1修正・護衛の防御値+2 |

## 5-4　怪異予兆カード

セッション開始前にGMが各PCへ1枚配布する。本人のみ閲覧可能。

カードには以下が書かれている：
- **公開条件**：いつカードを公開するか（例：「解明鍵2を入手した時」「哀愁メーターが5以上の時」）
- **公開効果**：公開することで何が起きるか（例：「解明鍵+1追加入手」「浄化メーター+2」）
- **秘密情報**：そのPCのみが知る設定（過去の因縁・怪異との接触歴・NPCとの関係）

<a id="ch6"></a>
# CHAPTER 6　魔法システム

## 6-1　魔法とは

魔法は「世界のソースコードへの直接介入」だ。素養が必要で、訓練によって習得する。この世界では電気やガスと同等のインフラとして位置づけられており、資格制度と安全基準が存在する。

**大きな魔法を使えば使うほど、怪異を生む可能性が跳ね上がる。**
**魔法で怪異を倒すことと魔法で怪異を生むことは表裏一体だ。**

## 6-2　魔法判定

魔法行使は**術**による判定を行う。

- 成功すれば意図した効果を得る
- 失敗した場合、または大規模魔法を使った場合は**怪異誘発判定**が発生する

### 怪異誘発判定

魔法失敗時または大規模魔法使用後に GMが1d6を振る。

| 術ランク | 怪異発生の出目 | 発生確率 |
|---|---|---|
| D | 4以上 | 約50% |
| C | 5以上 | 約33% |
| B | 6のみ | 約17% |
| A以上 | 発生しない | 0% |

大規模魔法（GMの判断）は術ランクに関係なく、発生確率が1段階悪化する。
発生した場合、周辺に**五級〜四級の怪異**が生まれる。

## 6-3　魔法言語

魔法使いは通常1つの言語で魔法を行使する。どの言語でも同じ結果を得られるが、アプローチが異なる。言語ごとに得意な状況が存在する。

| 言語 | 色 | アプローチの質感 | 得意な状況 |
|---|---|---|---|
| **P** | — | 汎用基礎言語。全言語と組み合わせ可能 | 魔導具の製造・調整 |
| **Igniscript** | 赤 | 燃やす・爆発・熱変容。力任せで直線的 | 短期決戦・突破 |
| **Lupis Surf** | 青 | 流す・包む・圧力で押し動かす。柔軟で広範囲 | 範囲制御・支援 |
| **Ivyo** | 緑 | 育てる・自然サイクルに乗せる。即効性より持続性 | 長期維持・回復補助 |
| **NGT** | 黄 | 加速・電気的処理・情報として解析する | デジタル機器との連携・調査補助 |
| **Monyx** | 無色 | 最小限の術式で組む。どんな方向にも転用できる | 魔導具調整・封印処理 |
| **P:** | 紫 | 弱体化・妨害・封印。Pの派生 | 怪異の封じ込め・弱体化 |
| **P'** | 桃 | 回復・強化・修復。Pの派生 | 医療補助・能力強化 |

*P: と P' は P の習得が前提。他の五言語は独立して習得できる。*

### 言語ごとの機械的差異

各言語は同じ術判定を行うが、以下の状況で+1修正を得る：

| 言語 | +1修正を得る状況 |
|---|---|
| Igniscript | 単体目標・短距離・集中火力が必要な場面 |
| Lupis Surf | 複数目標・範囲効果が必要な場面 |
| Ivyo | 持続時間が重要な場面（1シーン以上の効果） |
| NGT | 電子機器・通信・情報網への干渉 |
| Monyx | 既存の魔導具の調整・最小コストでの効果 |
| P: | 怪異のルールへの直接干渉・封印 |
| P' | ダメージや状態の回復・能力値の一時強化 |

# CHAPTER 7　異能と侵食率

## 7-1　異能とは

異能は「内側から生まれる力」だ。怪異と同じ発生原理を持ち、魔法とは根本的に異なる。

| 項目 | 魔法 | 異能 |
|---|---|---|
| 原理 | 外部（世界への介入） | 内部（自身の変質） |
| 再現性 | 高い | 低い |
| 習得方法 | 訓練と素養 | 体験と執着 |
| リスク | 怪異を生む | 使用者が変質する |

## 7-2　異能使用判定

異能の発動は**魂**による判定を行う。

| 結果 | 効果 | 侵食率上昇 |
|---|---|---|
| スペシャル（6） | 効果が最大出力で発揮。渇望メーター+2 | +5% |
| 成功（4〜5） | 効果発揮。渇望メーター+1 | +5〜10% |
| 失敗（2〜3） | 効果なし。渇望メーター+1 | +10% |
| ファンブル（1） | 制御不能な効果発動（GM決定）。渇望メーター+2・恐怖メーター+2 | +20% |

## 7-3　侵食率トラック

侵食率は 0〜100% で管理する。**侵食率は絶対に下がらない。一度踏み込んだ道は戻れない。**

| 侵食率 | 段階名 | 効果 |
|---|---|---|
| 0〜25% | **正常** | 影響なし |
| 26〜50% | **変容の兆し** | 外見に軽微な変化。怪異の「声」や「視点」が断片的に聞こえ始める。視野判定に+1 |
| 51〜75% | **半化の影** | 目に見える異形化。怪異の一部の能力が扱えるようになる（GMが決定）。毎場面で魂判定が発生 |
| 76〜99% | **臨界** | 重度の異形化。強力な異能が使用可能。GMは毎場面で「引き」を試みる |
| 100% | **怪異化** | キャラクター終了。PCはGMが管理する怪異になる |

## 8-1　作成の流れ

```
① 全能力値をランクDからスタート（7能力値すべてD）
② 背景を1つ選ぶ（2能力値がランクCへ昇格・初期効果あり）
③ クラスを1つ選ぶ（1能力値がランクCへ昇格 or 特定効果）

⑤ 覚醒パターンを選ぶ（先天型・ショック型・実験型・接触型）
⑥ ギフトを初期ギフトの中から1つ選ぶ
⑦ 装備をCPの範囲内で選択
⑧ 信念を設定（最大5点からスタート）
```

# CHAPTER 9　所属システム

所属は「使えるリソース」と「縛り」の両方を決定する。

## 祓部（はらえべ）

公的な怪異対処組織。魔導省の内部組織。

**強み：**
- **データベースアクセス：** 識を使う調査判定に +2（1セッションに3回まで）
- **援軍要請：** 1セッション1回、NPC祓士の支援を呼べる
- **古い怪異の知識：** 古怪班配属なら、一級・二級の古い怪異への識判定に追加 +1

**制約：**
- 任務命令への服従が義務。独断行動は処分対象
- 装備と行動に法的・組織的な制限がかかる
- 討伐派 vs 封印派の方針対立が任務中に影響することがある

## 傭兵

企業または個人として依頼を受けて動く討伐者。

**強み：**
- **装備グレード：** キャラクター作成時に装備を1ランク上から選べる（武装型スタートなら半装身型で始められる、など）
- **コネクション：** 縁による調査・情報収集判定（判や識で企業コネを使う場合）に +2（1セッションに3回まで）
- **二つ名：** 初期二つ名を1つ持つ。その分野の判定に常時 +1

**制約：**
- 収益がなければ活動継続が困難
- バック企業の方針に縛られる場合がある
- 法的に保護される範囲が限られる

## 無所属

国にも企業にも属さない個人または小集団。

**強み：**
- **生存者の勘：** 視野判定に常時 +1（場の空気・怪異の気配を読む経験値）
- **裏のルート：** 1セッション1回、通常では入手不可能な曰く付き装備・違法情報にアクセスできる
- **改造個体センサー：** 「何かがおかしい」と感じた時、GMに「改造個体か別個体か」のヒントを1つ要求できる（1セッション1回）

**制約：**
- 法的保護がない。討伐行為が犯罪として問われることがある
- 祓部・傭兵・謎の組織のどこからも警戒される
- 安定した補給・装備整備のルートがない

# APPENDIX　クイックリファレンス

## ■ 判定

1. 能力値ランクに応じたd6を振る（D=1個〜A=4個）
2. 1個を達成値、1個を共鳴ダイスとして選ぶ
3. 達成値+修正≧4で成功。6=スペシャル、1=ファンブル

## ■ 共鳴メーター上昇

- 判定の文脈に応じたメーターが上昇
- 通常+1、スペシャル+2、ファンブルは恐怖+2（強制）
- 臨界（10点）で代償発動→0にリセット（浄化のみ恩恵）

## ■ 魔法リスク（怪異誘発判定）

術判定失敗時または大規模魔法時にGMが1d6：
- D=4以上で怪異発生 / C=5以上 / B=6のみ / A以上=発生しない

## ■ 異能リスク（侵食率上昇）

- スペシャル：+5% / 成功：+5〜10% / 失敗：+10% / ファンブル：+20%
- 侵食率は絶対に下がらない

## ■ 戦闘（核護衛戦）の流れ

```
① 行動宣言（全員同時）
② 疾ランク順に解決（高→低→護衛→核）
③ 護衛への連鎖ダメージ判定（耐久値以上→核に同点伝播）
④ 討伐クロック更新
⑤ 制限ラウンドチェック
```

## ■ 解明完了宣言の効果

- 4鍵：クロック−1先行＋ラウンド延長権
- 3鍵：通常
- 2鍵以下：全判定−1＋護衛防御+2
