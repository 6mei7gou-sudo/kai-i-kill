# 怪異キル TRPG — コアメカニクス v1.0

> 「知識が命を救う。無知が人を殺す。」

調査なしの討伐は自殺行為だ。このゲームの本質は「戦うこと」より「知ること」にある。

---

## 1. 基本ルール

### ダイスシステム：2d6 + 属性

全ての判定は **2d6 + 関連属性値** で解決する。

| 合計値 | 結果 | 意味 |
|--------|------|------|
| 2〜8 | **失敗** | 目標は達成されず、GMが何らかの代償を与える |
| 9〜11 | **部分成功** | 目標は達成されるが、コスト・代償・条件が伴う |
| 12〜13 | **成功** | 目標を達成する |
| 14以上 | **完全成功** | 目標を達成し、追加の利得を得る |

---

## 2. 属性（6属性）

キャラクターは以下の6つの属性を持つ。

| 属性 | 読み | 用途 |
|------|------|------|
| **視野** | しや | 現場調査・物証収集・怪異の気配察知 |
| **識** | しき | 資料調査・データベース照会・民俗学的知識・怪異の識別 |
| **体** | たい | 直接戦闘・身体的耐久・逃走 |
| **術** | じゅつ | 魔法行使・魔導具運用・装備操作 |
| **魂** | こん | 精神的耐久・異能制御・侵食への抵抗・恐怖への抵抗 |
| **縁** | えん | 聞き込み・人脈活用・評判・組織との関係維持 |

### キャラクター作成：属性の配分

**18点を6属性に配分する。各属性の最小値は1、最大値は5。**

推奨配分例（プレイヤーがスタイルを決めやすいよう）：
- 専門家型：主属性5・副属性4・その他1〜2
- バランス型：3・3・3・3・3・3
- 二刀型：5・4・3・2・2・2

## 3. 侵食率（いのうトラック）

異能を使うたびに、使用者は怪異に近づいていく。侵食率は 0〜100% で管理する。

| 侵食率 | 段階名 | 効果 |
|--------|--------|------|
| 0〜25% | **正常** | 影響なし |
| 26〜50% | **変容の兆し** | 外見に軽微な変化。怪異の「声」や「視点」が断片的に聞こえ始める |
| 51〜75% | **半化の影** | 目に見える異形化。怪異の一部の能力が扱えるようになる。定期的に魂判定が発生 |
| 76〜99% | **臨界** | 重度の異形化。強力な異能が使用可能。GM は毎場面で「引き」を試みる |
| 100% | **怪異化** | キャラクター終了。PCはGMが管理する怪異になる |

侵食率は「使用した異能のレベル」に応じて変動する（詳細は異能ルールを参照）。
侵食率を下げる方法は存在しない。一度踏み込んだ道は戻れない。

## 4. セッション構造：三段階討伐プロセス

全てのシナリオは三幕構成で進行する。この順序を飛ばした討伐は自殺行為だ。

---

### 第一幕：調査フェーズ

怪異の「情報ボード」を埋めることが目的だ。

#### 情報ボードの三要素

| 要素 | 内容 | 主な調査手段 |
|------|------|------|
| **正体** | この怪異は何か？噂の起源は何か？いつ生まれたか？ | 識・縁 |
| **核** | 怪異の核はどこに宿っているか？ | 視野・識 |
| **ルール** | この怪異が持つ行動・回避・攻撃の法則は何か？ | 視野・識・縁（危険を冒して） |

各要素を「判明」にするには、少なくとも1回の成功した判定が必要。
調査は複数の手段を組み合わせることで精度が上がる。

#### 改造個体・別個体の罠

文献との齟齬、被害パターンの不一致、核の所在の混乱——これらは「何かがおかしい」サインだ。
改造個体の解明中に「ある一点でルールが食い違う瞬間」が来る。
最も危険なパターン：**別個体と判明した瞬間、これまでの解明が全て無効になる。**

---

### 第二幕：解明フェーズ

情報が揃ったら、**暴露判定**を行う。

#### 暴露判定（識 または 視野）

| 情報ボードの状況 | 修正 |
|------|------|
| 正体のみ判明 | −2 |
| 正体＋核 判明 | ±0 |
| 3要素すべて判明 | +2 |

**成功（12以上）：**「暴露ウィンドウ」が開く。次の討伐フェーズ全体を通じて怪異の弱点攻撃に +3 のボーナスを得る。
**部分成功（9〜11）：**ウィンドウは開くが、怪異も警戒を高める（GMが一つ追加行動を行う）。
**失敗（8以下）：**怪異が「看破された」と感知し行動を変える。GMは怪異の行動に1つ変更を加える。

---

### 第三幕：討伐フェーズ

知識を武器にして核を破壊する。

#### 攻撃の修正

| 状況 | 修正 |
|------|------|
| 怪異のルールを利用した攻撃 | ±0（標準） |
| 暴露ウィンドウ中の弱点攻撃 | +3 |
| ルールを無視した攻撃 | −3 |
| ルール依存型怪異へのルール無視攻撃 | 無効 |

#### 核へのダメージ

攻撃判定の結果に応じて核のHPを削る。

| 判定結果 | ダメージ | 備考 |
|----------|----------|------|
| 部分成功（9〜11） | 1点 | ただし何らかのコストを伴う（反撃・消耗・怪異のルール変化など） |
| 成功（12〜13） | 2点 | 標準的なダメージ |
| 完全成功（14以上） | 3点 | ボーナスあり（GMと相談） |
| 暴露ウィンドウ中 | +1点 | 全ての攻撃に加算 |

**核のHPが0になると怪異は消滅する**（一時型）または制圧状態になる（循環型・定着型）。

核を物理的・霊的・魔法的手段で破壊することが基本的な討伐手段だ。
核の場所と無効化手段は怪異によって異なる。
**戦闘力だけで解決できる怪異は存在しない。**

## 5. 魔法システム

魔法は「世界のソースコードへの直接介入」だ。素養が必要であり、訓練によって習得する。

### 魔法判定：2d6 + 術

成功すれば意図した効果を得る。失敗した場合、または大規模魔法を使った場合は **怪異誘発判定** が発生する。

#### 怪異誘発判定

魔法使用後に GMが2d6を振る：

| 条件 | 結果 |
|------|------|
| 出目 ≤ 術値 | **安全**（怪異は発生しない） |
| 出目 > 術値 | **周辺に小型怪異（五級〜四級）が発生** |

**確率の目安（術値別）：**

| 術値 | 怪異発生の確率 |
|------|------|
| 2 | 約83% |
| 3 | 約58% |
| 4 | 約42% |
| 5 | 約28% |

術値が低いキャラクターが大きな魔法を使うのは命がけだ。
大規模魔法（GM判断）は術値に関わらず発動する。

大きな魔法を使えば使うほど、怪異を生む可能性が跳ね上がる。**魔法で怪異を倒すことと魔法で怪異を生むことは表裏一体だ。**

### 魔法言語

魔法使いは通常1つの言語で魔法を行使する。どの言語でも同じ結果を得られるが、アプローチが異なる。

| 言語 | 色 | アプローチの質感 |
|------|-----|------|
| **P** | — | 汎用基礎言語。全言語と組み合わせ可能。魔導具の基幹 |
| **Igniscript** | 赤 | 燃やす・爆発・熱変容。力任せで直線的 |
| **Lupis Surf** | 青 | 流す・包む・圧力。柔軟で広範囲 |
| **Ivyo** | 緑 | 育てる・自然サイクル。即効性より持続性 |
| **NGT** | 黄 | 加速・電気的処理・情報解析。デジタルインフラと親和 |
| **Monyx** | 無色 | 最小術式・汎用転用。魔導具調整の主力 |
| **P:** | 紫 | 弱体化・妨害・封印。Pの派生。習得者が少ない |
| **P'** | 桃 | 回復・強化・修復。Pの派生。身体的負荷が大きい |

*P: と P' は P の習得が前提。他の五言語は独立して習得できる。*

## 6. 異能システム

異能は「内側から生まれる力」だ。怪異と同じ発生原理を持ち、魔法とは根本的に異なる。

### 異能の特性

| 項目 | 魔法 | 異能 |
|------|------|------|
| 原理 | 外部（世界への介入） | 内部（自身の変質） |
| 再現性 | 高い | 低い |
| 習得方法 | 訓練と素養 | 体験と執着 |
| リスク | 怪異を生む | 使用者が変質する |

### 異能使用判定：2d6 + 魂

| 結果 | 効果 |
|------|------|
| 成功（12以上） | 効果発揮。侵食率 +5〜10%（異能レベルに応じる） |
| 部分成功（9〜11） | 効果発揮。侵食率 +10〜15% |
| 失敗（8以下） | 効果なし。侵食率 +10% |
| 暴走（2d6の目が2） | 制御不能の効果発動。侵食率 +20%。GMが内容を決定 |

### 侵食率と異能の強化

侵食率が高いほど、より強力な異能効果を発動できる可能性がある。これは「力の代償」だ。
侵食率 51% 以上では、怪異の一部の特性が使用可能になるが、精神の独立性が揺らぎ始める。

## 7. 装備システム

装備は六種に分類される。全ての装備に魔導具を組み込むことが可能だが、規模に比例して怪異発生リスクが生まれる。

| 種別 | 主用途 | 戦闘修正 | 怪異リスク | 扱いやすさ |
|------|--------|----------|-----------|-----------|
| **武装型** | 近中距離戦闘 | +1 | 低 | 誰でも使える |
| **独立型** | 偵察・支援・包囲 | 援護+1 | 中 | 機体依存 |
| **半装身型** | 部位強化・中火力 | 攻防+1 | 中 | 中程度 |
| **全装身型** | 高火力・高防御 | 攻防+2 | 高 | 扱える人間が少ない |
| **搭乗型** | 機動・輸送・制圧 | 機動力+2 | 中 | 中程度 |
| **戦闘用搭乗型** | 大型怪異討伐 | 三級以下に+3 | 非常に高 | 個人運用は違法 |

**全装身型の都市伝説化リスク：** 一般市民に目撃されると都市伝説として拡散しやすく、新たな怪異を生む火種になる。
**戦闘用搭乗型の使用制限：** 個人が私有・運用することは法律で禁止されている。

## 8. 所属システム

キャラクターの所属は「使えるリソース」と「縛り」の両方を決定する。

### 祓部（はらえべ）

公的な怪異対処組織。魔導省の内部組織。

**強み：**
- **データベースアクセス：** 識を使う調査判定に +2（1セッションに3回まで）
- **援軍要請：** 1セッション1回、NPC祓士の支援を呼べる
- **古い怪異の知識：** 古怪班配属なら、一級・二級の古い怪異への識判定に追加 +1

**制約：**
- 任務命令への服従が義務。独断行動は処分対象
- 装備と行動に法的・組織的な制限がかかる
- 討伐派 vs 封印派の方針対立が任務中に影響することがある

---

### 傭兵

企業または個人として依頼を受けて動く討伐者。

**強み：**
- **装備グレード：** キャラクター作成時に装備を1ランク上から選べる（例：武装型スタートなら半装身型、半装身型なら全装身型で始められる）
- **コネクション：** 縁を使う調査・情報収集判定に +2（1セッションに3回まで。企業・傭兵コミュニティへの繋がりを使う）
- **二つ名：** 初期二つ名を1つ持つ。その分野の判定に常時 +1（例：「封印の名手」なら封印処理に+1）

**制約：**
- 収益がなければ活動継続が困難（補給・装備整備のコストがかかる）
- バック企業の方針に縛られる場合がある
- 法的に保護される範囲が限られる

---

### 無所属

国にも企業にも属さない個人または小集団。

**強み：**
- **生存者の勘：** 視野判定に常時 +1（場の空気・怪異の気配を読む経験値）
- **裏のルート：** 1セッション1回、通常では入手不可能な曰く付き装備・違法情報にアクセスできる
- **改造個体センサー：** 「何かがおかしい」と感じた時、GMに「改造個体か別個体か」の判断ヒントを1つ要求できる（1セッション1回）

**制約：**
- 法的保護がない。討伐行為が犯罪として問われることがある
- 祓部・傭兵・謎の組織のどこからも警戒される
- 安定した補給・装備整備のルートがない

## 9. 怪異データブロック（GM向け）

全ての怪異は以下の形式で定義される。

```
怪異名：[名前]
存在強度等級：[特級 / 一級 / 二級 / 三級 / 四級 / 五級]
脅威度種別：[甲種 / 乙種 / 丙種 / 丁種]
永続性：[一時型 / 循環型（都市伝説型） / 定着型（禁足地・土地型）]
影響範囲：[広域型 / 局所型]
被害性質：[物理型 / 精神型 / 複合型]

核：[場所・HP・耐性]

ルール（1〜5個）：
  - ルール1：[内容]
  - ルール2：[内容]
  ...

耐性：[物理耐性型 / 霊的耐性型 / ルール依存型 / なし]

調査情報：
  正体：[どんな調査で、何が判明するか]
  核の場所：[どんな調査で、何が判明するか]
  ルール：[どんな調査で、何が判明するか]

改造・別個体リスク：[ある / なし / 教団関与の可能性]
```

### 怪異等級の目安

| 等級 | HP目安 | 特記 |
|------|--------|------|
| 五級 | 5〜10 | 自然消滅することもある |
| 四級 | 10〜20 | 新しい怪異。ルールが単純 |
| 三級 | 20〜35 | 拡散が進んだ怪異。ルール2〜3個 |
| 二級 | 35〜60 | 記録のある古い怪異。ルール3〜4個 |
| 一級 | 60〜100 | 長い歴史を持つ強力な怪異。ルール4〜5個 |
| 特級 | 100以上 | 消滅困難。封印・管理が限界 |

## 10. 覚醒パターン

キャラクターが怪異対処者になった経緯。動機と背景に直結する。

| パターン | 概要 | 傾向 |
|----------|------|------|
| **先天覚醒型** | 生まれつき素養を持ち訓練で開花 | 祓部に多い。技術の安定性が高い |
| **ショック覚醒型** | 怪異に関わる強烈な体験が引き金 | 恨みや喪失が動機に直結しやすい |
| **実験覚醒型** | 人体実験で強制覚醒 | 素養強度が高いが身体への代償が大きい |
| **接触覚醒型** | 怪異の核や特殊素材への長期接触で覚醒 | 怪異への感知能力が高い。変質リスクを伴う |

---

## 11. キャラクター作成の手順

1. **所属を選ぶ：** 祓部・傭兵・無所属のどれか
2. **覚醒パターンを選ぶ：** 先天型・ショック型・実験型・接触型のどれか
3. **属性を決める：** 18点を6属性に配分（各1〜5）
4. **能力スタイルを選ぶ：** 魔法（どの言語か）か、異能か、またはその両方か
5. **主力装備を選ぶ：** 六種の中から選択（傭兵は1段階グレードアップ）
6. **因縁を決める：** 何を失ったか、何を追っているか。この世界で戦い続ける理由

---

## 12. 生き延びるための原則

この世界の討伐者が受け継いできた言葉だ。

> 調査なしの討伐は自殺行為だ
> 解明なしの討伐は自殺行為だ
> ルールを破れば破るほど逃げられなくなる
> 強さだけで解決できる怪異は存在しない
> 知識が命を救う。無知が人を殺す
> 話しても誰も信じてくれない。それがこの仕事だ
