# CHAPTER 0　このゲームについて

## 0-1　概要

舞台は近未来の架空日本。魔法はインフラとして社会に組み込まれ、その裏側では噂が質量を持ち、現実にバグとして侵食する怪異が日常的に発生している。

あなたは討伐者だ。怪異を調査し、核とルールを解明し、討伐する。

このゲームには二つの大きな特徴がある。

**① 《共鳴記録》システム**
能力値のランク（D〜S）に応じてダイスの数が変わる。多くのダイスを振るほど成功率は上がる。しかし判定のたびに、その行為に紐づいた感情の波が討伐者の体内で共鳴し、**感情共鳴メーター**に積み重なっていく。共鳴が深まればギフトが解放されるが、特定の感情が臨界まで達すると代償が訪れる。

**② 怪異予兆カード**
GMはセッション開始前に各PCへ「怪異予兆カード」を1枚配る。そこに書かれた情報は本人のみが知る秘密だ。カードに書かれた条件を満たすと公開され、物語に新たな次元が加わる。

## 0-2　面白さの三層構造

| 層 | 体験の核心 |
|---|---|
| **パズルとしての討伐** | 怪異のルールと核を解明して初めて倒せる。知識が命を救う。 |
| **感情のジレンマとしての判定** | 強く行動するほど感情が共鳴する。共鳴は力だが、特定の感情に偏ると代償が来る。 |
| **問いとしての世界観** | 怪異と神の違いは恐れか信仰か。討伐者たちは正しいことをしているのか。 |

# CHAPTER 1　世界観

## 1-1　怪異のメカニズム

怪異とは、集合的な噂・言説・信念が臨界点を超えた時に現実へと侵食するバグだ。全ての怪異は**核（コア）**と**ルール**を持つ。核を破壊することで怪異を消滅させられるが、それは調査と解明を経て初めて判明する。

| 概念 | 内容 |
|---|---|
| **核（コア）** | 怪異の存在の中心。物・場所・記録媒体に宿る。稀に人間に宿る。 |
| **ルール** | 怪異が人を害する条件。ルールを破るほど捕捉される。 |

## 1-2　禁足地と公的封鎖

特定の土地に根を張った定着型怪異は、核が土地と一体化しており通常手段では無効化できない。このような場所を**禁足地（きんそくち）**と呼ぶ。

祓部は禁足地を物理的・魔法的に封鎖し、定期的な状態確認を行う。禁足地は消えも滅びもしない。ただ管理される。

> **「感染域」という隔絶概念はこのシステムに存在しない。**
> 怪異との対峙は現実の路地、廃ビル、駅構内、深夜の公園で起きる。
> 場所が変容するのではなく、人間が変容する。

## 1-3　能力体系

| 種別 | 概要 | リスク |
|---|---|---|
| **魔法（チートコード）** | 世界のソースコードへの直接介入。訓練で習得。 | 使用規模に比例して怪異発生リスクが上がる。 |
| **異能（アプリケーション）** | 怪異と同じ発生原理の延長線上にある力。体験・執着・信念から発現。 | 使用のたびに侵食率が蓄積する。 |

## 1-4　三組織

| 組織 | 概要 | 特性 |
|---|---|---|
| **祓部（はらえべ）** | 魔導省内部の公的討伐機関。神職系称号を持つ専門家集団。 | データベース・情報優位。禁足地の封印管理が業務。 |
| **傭兵集団** | 独立系企業バックの民間討伐力。二つ名持ちの個人特化型も存在。 | 装備性能・戦闘技術優位。実験覚醒型の元教団実験体を保護する企業も。 |
| **無所属** | 国にも企業にも属さない個人または小集団。法的保護なし。 | 曰く付き装備・怪異核素材魔導具・独自情報網。 |

# CHAPTER 2　基本判定システム

## 2-1　能力値とランク

キャラクターは6つの能力値を持つ。各能力値はランクで表される。

| 能力値 | 略称 | 主な用途 |
|---|---|---|
| **フィジカル** | FIZ | 格闘・突破・耐久 |
| **スピード** | SPD | 先手・回避・追跡 |
| **ナレッジ** | KNW | 調査・解明・文献 |
| **ジャッジメント** | JDG | 解明宣言・看破・戦術 |
| **センス** | SNS | 怪異感知・証言聴取 |
| **ウィル** | WIL | 侵食抵抗・信念維持 |

| ランク | ダイス数 | 特徴 |
|---|---|---|
| **D** | 1d6 | 共鳴なし |
| **C** | 2d6 | 余り1個→共鳴へ |
| **B** | 3d6 | 余り2個→共鳴へ |
| **A** | 4d6 | 余り3個→共鳴へ |
| **S** | 4d6+固有特典 | 余り3個→共鳴へ＋追加効果 |

## 2-2　判定手順

1. GMが要求する能力値を宣言
2. ランクに応じたd6を振る
3. **1個を「達成値ダイス」として選択**
4. **残りの中から1個を「共鳴ダイス」として選択**（残り0個=選択不要）
5. 達成値（出目＋修正）が**4以上で成功**

| 出目 | 効果 |
|---|---|
| **6** | **スペシャル**：自動成功＋追加効果 |
| 4〜5 | 成功 |
| 2〜3 | 失敗 |
| **1** | **ファンブル**：自動失敗＋ファンブル表を参照 |

# CHAPTER 3　《共鳴記録》システム

## 3-1　設計思想

怪異と戦い続ける討伐者の体は、少しずつ怪異の波長に共鳴していく。それは能力として現れることもあるし、人間としての何かを削ることもある。**《共鳴記録》はその蓄積を記録するシステムだ。**

アマデウスの「インガ（余剰ダイスを運命の輪へ配置）」という概念を参照しているが、構造は根本的に異なる。各PCが**個別に**6種の感情共鳴メーターを持ち、共鳴ダイスの出目ではなく**判定の内容と文脈**によってどのメーターが上昇するかが決まる。

## 3-2　6種の感情共鳴メーター

各メーターの最大値は**10点**。各メーターが蓄積するごとに覚醒段階が変化する。

| メーター | 色 | 上昇する判定の文脈 | 臨界の代償 |
|---|---|---|---|
| **恐怖（FEAR）** | 暗赤 | 逃走・退避・防御に失敗した時、怪異のルールを破った時 | 次の【WIL】判定が自動ファンブル。侵食率+10% |
| **怒り（RAGE）** | 赤橙 | 攻撃判定スペシャル時、仲間が傷ついた時の行動 | 1ラウンド行動が強制：最も近い敵への最大火力攻撃のみ |
| **哀愁（SORROW）** | 青 | 解明鍵を1つ入手した時、NPCとの別れの場面 | 次の調査判定がすべて−1。「信念ポイント」1消費 |
| **焦燥（HASTE）** | 橙 | 制限ラウンド残り1で行動した時、手がかりを失った時 | 次の行動宣言を相手より先に強制公開（奇襲不可） |
| **渇望（THIRST）** | 紫 | 異能・怪異核素材装備を使用した時、侵食ロール後 | 侵食率+5%。次の侵食ロールの失敗閾値が1段階悪化 |
| **浄化（PURGE）** | 白緑 | 解明完了宣言成功時、信念ポイントを全消費した時 | 代償なし（浄化のみ臨界に上限なし。10点でリセット＋恩恵） |

## 3-3　共鳴メーターの上昇タイミング

共鳴ダイスを選択した後、**その判定の文脈**を参照して上昇するメーターを決める。

- **共鳴ダイスの出目は関係しない**（出目はGMへの参考情報として見せるだけ）
- 文脈が複数のメーターに該当する場合、PLが1つ選ぶ
- ランクDの判定（ダイス1個）は共鳴ダイスなし→共鳴メーター上昇なし

### 共鳴上昇量

| 状況 | 上昇量 |
|---|---|
| 通常の判定（成功・失敗問わず） | 対応メーター+1 |
| スペシャル（出目6）の判定 | 対応メーター+2 |
| ファンブル（出目1）の判定 | 恐怖メーター+2（強制） |

## 3-4　覚醒段階

メーター内の蓄積点数で覚醒段階が決まる。段階に応じてギフトが解放される。

| 蓄積点 | 覚醒段階 | 解放されるギフト |
|---|---|---|
| 1〜3 | **第一段階** | 初級覚醒ギフト |
| 4〜6 | **第二段階** | 中級覚醒ギフト |
| 7〜9 | **第三段階** | 上級覚醒ギフト |
| 10 | **臨界（CRITICAL）** | 代償発動（浄化のみ恩恵） |

## 3-5　臨界後の処理

- **恐怖・怒り・哀愁・焦燥・渇望**：臨界の代償を受けた後、そのメーターを**0にリセット**
- **浄化**：10点で「大浄化」ギフトを発動し0にリセット（代償なし）
- リセット後は再び積み上げが始まる

## 3-6　PLへの開示ルール

全員の共鳴記録メーターは卓全体に公開される。他のPCの感情状態が見えることで、物語的な連動が生まれる。

# CHAPTER 4　怪異との戦闘《核護衛戦》

## 4-1　設計思想

アマデウスの「脅威（シールド）＋本体」「ラウンド数制限」「高ダメージレース」という構造をKAI-I//KILLの世界観に移植した戦闘システム。

怪異の核は剥き出しでない。核には**護衛（ガーディアン）**と呼ばれる存在が取り巻いている。護衛を突破せずに核を攻撃することは、通常不可能だ。

**討伐の本質は「護衛突破＋核へのダメージ」を制限ラウンド内に完了させること。**

## 4-2　核護衛戦の基本構造

### 戦闘の構成要素

```
【核（コア）】
  ↑  核を守る
【護衛1】 【護衛2】 【護衛3】（怪異の等級によって数が変わる）
  ↕  PCたちが対峙する
【討伐者チーム】
```

### 制限ラウンド（タイムリミット）

戦闘には**制限ラウンド**が設けられる。ラウンドを超過すると、怪異が「核強化フェイズ」に移行し、護衛が完全回復・核の防御値が上昇する。

| 怪異等級 | 制限ラウンド | 護衛の数 |
|---|---|---|
| 五級 | 3ラウンド | なし（核直接） |
| 四級 | 3ラウンド | 護衛1体 |
| 三級 | 3ラウンド（+1回延長可） | 護衛2体 |
| 二級 | 4ラウンド | 護衛2〜3体 |
| 一級 | 4ラウンド（+1回延長可） | 護衛3体 |
| 特級 | **封印クロック方式**（後述） | 護衛多数＋特殊ギミック |

### 延長の条件

解明完了宣言が成立している場合、「制限ラウンド+1」の延長を**1回だけ**適用できる。

## 4-3　護衛（ガーディアン）

護衛は怪異の核から生み出された存在だ。核と同じルールを部分的に持つ。

### 護衛データの構成

| 項目 | 内容 |
|---|---|
| **耐久値（HP）** | 護衛の体力。0で機能停止。 |
| **行動値** | 護衛の攻撃・妨害の強度 |
| **特性** | 護衛固有のルール（後述） |
| **連鎖条件** | 1ラウンドで耐久値以上のダメージを与えた場合の特殊効果 |

### 連鎖ダメージ（アマデウス「脅威」の移植）

護衛に対して**1ラウンド内で耐久値以上のダメージ**を与えると：
1. 護衛は**そのラウンド終了まで行動不能**になる
2. 護衛の耐久値と**同点のダメージ**が核に自動的に伝播する
3. スペシャル（出目6）による攻撃で達成した場合、伝播ダメージが**×1.5（端数切り上げ）**

> **例：** 耐久値5の護衛に7ダメージを与えた場合→護衛行動不能＋核に5ダメージ自動伝播。スペシャルで達成なら核に8ダメージ。

### 護衛特性の例

| 特性名 | 効果 |
|---|---|
| **連結（CHAIN）** | この護衛が行動不能になると隣の護衛のHP回復3 |
| **干渉（BLOCK）** | この護衛がいる限り、核への直接攻撃は−3修正 |
| **反射（REFLECT）** | ダメージの半分（切り捨て）が攻撃者に跳ね返る |
| **再生（REGEN）** | ラウンド開始時にHP+2回復 |
| **感染（INFECT）** | この護衛からダメージを受けると恐怖メーター+1 |

## 4-4　行動順（イニシアチブ）

各ラウンドの行動順は**SPDランクによる宣言制**で行う。

1. GMが「行動宣言フェイズ」を開始
2. PLは行動順に関係なく全員同時に行動を宣言
3. **SPDランクが高いPC→低いPC→護衛→核**の順で行動を解決
4. 同ランクの場合、PLが望む順番で解決（PLに有利なルール）

### 行動の種類

| 行動 | 内容 | 使用能力値 |
|---|---|---|
| **攻撃（ATTACK）** | 護衛または核へのダメージ行動 | FIZ/SPD（武器による） |
| **援護（COVER）** | 別のPCへの攻撃を肩代わりする | FIZ |
| **解明（DECODE）** | 戦闘中に解明鍵を進める（戦闘フェイズ限定） | KNW/SNS |
| **干渉（DISTURB）** | 護衛の特性を一時的に無効化する | JDG |
| **回避専念（EVADE）** | その場からの完全回避。攻撃不可だがダメージ無効。 | SPD |
| **浄化（CLEANSE）** | WILランクで判定。成功で自分の共鳴メーター1つ−2点 | WIL |
| **異能発動（ABILITY）** | 異能を使用（侵食ロール発生） | 異能種別 |

## 4-5　ダメージ計算

```
ダメージ = 達成値 + 武器修正 − 対象の防御値
```

| 達成値 | 武器なし | 武器あり（修正値は装備による） |
|---|---|---|
| 4 | 1 | 1+武器修正 |
| 5 | 2 | 2+武器修正 |
| 6（スペシャル） | 3+特典 | 3+武器修正+特典 |

護衛の防御値は怪異データに記載。核の防御値は護衛が存在する限り高い。

### 核への直接攻撃

解明完了宣言を経た後のみ、**核への直接攻撃が可能**になる。護衛が全て行動不能または破壊されている場合も直接攻撃可能。

核への直接攻撃が成功すると討伐クロックが進行する（後述）。

## 4-6　護衛の行動

護衛はGMが制御する。ラウンドごとに以下から行動を選ぶ。

| 行動 | 効果 |
|---|---|
| **攻撃** | 任意のPCに行動値分のダメージ |
| **妨害** | 任意のPCの次の判定に−1修正 |
| **庇い** | 核への攻撃を自分で受ける（1ラウンドに1回） |
| **強化** | 別の護衛のHPを3回復 |

## 4-7　討伐クロックと終了条件

戦闘の勝利条件は、討伐クロックを**0にすること**だ（クロックは解明パートで設定）。

| 怪異等級 | 初期クロック値 |
|---|---|
| 五級 | 4 |
| 四級 | 6 |
| 三級 | 8 |
| 二級 | 10 |
| 一級 | 12 |
| 特級 | 封印クロック（別途設定） |

クロックは以下の方法で進行する。

| 行動 | クロック進行 |
|---|---|
| 核への通常攻撃（成功） | −1 |
| 核への攻撃（スペシャル） | −2 |
| 護衛への連鎖ダメージ伝播 | −（伝播ダメージ分） |
| 解明完了後の浄化ギフト発動 | GMが設定（通常−2〜−4） |
| 核への直接攻撃（成功・解明完了後） | 残クロック全消化（即時討伐） |

### 特級の封印クロック

特級怪異は「破壊」が不可能な場合がある。この場合、討伐クロックの代わりに**封印クロック**を用いる。封印クロックを最大まで積み上げると、怪異は「管理封印状態」となりシナリオクリアとなる。

## 4-8　共鳴記録と戦闘

戦闘中も共鳴記録は機能する。特に頻繁に上昇するメーターを以下に示す。

| 戦闘状況 | 上昇するメーター |
|---|---|
| 攻撃判定（命中） | 怒り +1 |
| 攻撃判定（スペシャル命中） | 怒り +2 |
| ダメージを受けた時 | 恐怖 +1 |
| 護衛への連鎖ダメージ成功 | 怒り +2（戦果） |
| 制限ラウンド残り1 | 焦燥 +1（全PC強制） |
| 仲間が行動不能になった場面 | 哀愁 +1（任意）or 怒り +1（任意） |
| 異能を使用した場面 | 渇望 +1 |

<a id="ch5"></a>
# CHAPTER 5　調査・解明・討伐の三段階

## 5-1　概要

討伐は三段階で構成される。戦闘（第三段階）は出口だが、最初の二段階が生死を分ける。

```
[第一段階] 調査プロセス
    ↓ 解明鍵の収集
[第二段階] 解明プロセス
    ↓ 解明完了宣言（4鍵達成推奨）
[第三段階] 討伐プロセス《核護衛戦》
    ↓ 討伐クロック消化
[終了] 討伐完了・撤退・封印
```

## 5-2　四つの解明鍵

| 鍵 | 内容 | 主な判定 |
|---|---|---|
| **①怪異の正体** | 古い/新しい怪異か、元になった噂の内容 | KNW・SNS |
| **②核の所在** | 核が宿っている物・場所・人間 | SNS・KNW |
| **③被害パターン** | ルールの構造、誰が狙われるか | JDG・KNW |

# APPENDIX　クイックリファレンス

## ■ 判定

1. 能力値ランクに応じたd6を振る（D=1個〜A=4個）
2. 1個を達成値、1個を共鳴ダイスとして選ぶ
3. 達成値+修正≧4で成功。6=スペシャル、1=ファンブル

## ■ 共鳴メーター上昇

- 判定の文脈に応じたメーターが上昇
- 通常+1、スペシャル+2、ファンブルは恐怖+2（強制）
- 臨界（10点）で代償発動→0にリセット（浄化のみ恩恵）

## ■ 戦闘（核護衛戦）の流れ

```
① 行動宣言（全員同時）
② SPD順に解決（高→低→護衛→核）
③ 護衛への連鎖ダメージ判定
④ 討伐クロック更新
⑤ 制限ラウンドチェック
```

## ■ 護衛への連鎖ダメージ

1ラウンドで耐久値以上のダメージ→護衛行動不能＋核に耐久値分のダメージ伝播（スペシャルなら×1.5）
