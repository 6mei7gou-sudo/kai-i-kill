## C-1　サイバネティクスとは

魔法がインフラとして定着したこの社会では、身体の一部を魔導機関で置換・増強する「サイバネティクス処置」が半ば合法的に行われている。四肢の義肢から感覚器官の拡張、脳内に埋め込む情報端末まで、その範囲は広い。

ただし人体への不可逆な干渉は侵食率と深く関わる。怪異核素材を応用したサイバネティクスは性能こそ高いが、使用者を内側から別のものに変えていく。

正規のサイバネティクスは蒼鉄機工が祓部向けに供給する純正品と、雷禽重工が傭兵市場に出している高出力品が中心だ。鴉羽技研の闇施術、蜃気楼工廠の怪異核インプラントも存在するが、どちらも使用者を選ぶ。

### 装備との違い

| | 装備 | サイバネティクス |
|---|---|---|
| 着脱 | 可能 | 不可逆（一部を除く） |
| CPコスト | 装備CP内 | 専用の**改造CP**を使用 |
| 侵食リスク | 使用時のみ | 常時。等級に応じて基礎侵食が上昇 |
| 被弾時 | 装備が壊れる | 身体にダメージが直結 |

## C-2　改造等級と侵食

サイバネティクスは「どこまで自分の肉体を手放すか」で3つの等級に分かれる。

| 等級 | 名称 | 改造CP上限 | 基礎侵食加算 | 概要 |
|---|---|---|---|---|
| **Ⅰ** | 表層改造 | 3CP | +0% | コンタクトレンズ型HUD、皮下センサー、神経接続端子など。見た目には分からない |
| **Ⅱ** | 深層改造 | 6CP | +5% | 義肢、人工臓器、脳内チップ。服を脱げば分かる。日常生活に若干の不便 |
| **Ⅲ** | 全置換 | 10CP | +15% | 四肢全交換、脊髄魔導回路、全身装甲皮膚。もはや人間に見えない。社会的な障壁が生じる |

- 改造CPは装備CPとは別枠で管理する
- 基礎侵食加算はキャラクター作成時に初期侵食率に足す
- 等級Ⅲの討伐者は一般人から「怪異と間違われる」リスクがある

## C-3　改造部位

人体を6つの部位に分け、各部位に1つのサイバネティクスを搭載できる。同じ部位に2つ以上を搭載する場合は「過積載」として追加の拒絶判定が必要になる。

| 部位 | 改造の例 | 対応能力値 |
|---|---|---|
| **頭部** | HUDインプラント、脳内演算チップ、聴覚増幅器 | 識・視野 |
| **眼球** | サーマルアイ、怪異視覚フィルター、広域走査眼 | 視野・判 |
| **胴体** | 人工心肺、脊髄魔導回路、皮下装甲 | 体・魂 |
| **右腕** | 義腕、内蔵武器、グラップルアンカー | 体・術 |
| **左腕** | 義腕、シールドジェネレーター、感知アレイ | 体・術 |
| **脚部** | 脚部強化、跳躍ブースター、消音歩行機構 | 体・疾 |

## C-5　メーカー別サイバネティクス

### 蒼鉄機工の純正品

安全第一。拒絶反応が起きにくく、官製品タグ付きで法的に保護される。性能は控えめだが信頼性が高い。祓部の技術班で施術を受けられる。

- 特徴：拒絶判定に+1ボーナス。侵食加算が他社より1段階低い
- 制約：違法改造を加えた瞬間、官製品タグが破損して祓部に通報される

### 雷禽重工の高出力品

性能偏重。素養依存のブーストが多く、身体への負荷が大きい。傭兵の実戦データを施術にフィードバックしており、試作品テストプログラムの一環としてサイバネティクスの無償施術を行うこともある。

- 特徴：能力修正が高い。素養持ちへの追加ボーナスあり
- 制約：実戦データが自動送信される。拒絶リスクが高い

### 鴉羽技研の闇施術

裏の施術所で行われる非公式改造。違法だが細工の精度は高い。隠密性に優れた改造が多く、発覚しにくい設計が売り。

- 特徴：隠密・潜入向きの改造が豊富。検知されにくい
- 制約：保証なし。施術失敗のリスク。祓部に発覚すると逮捕

### 蜃気楼工廠の怪異核インプラント

怪異核素材を直接人体に埋め込む禁忌の施術。性能は異常に高いが、使用者を確実に人間から遠ざける。

- 特徴：ランク上昇や直感系の効果が多い。通常では得られない能力
- 制約：毎セッション拒絶判定。侵食加算が最大。「人間をやめる」覚悟が必要

<a id="c6"></a>
## C-6　拒絶反応と適応判定

サイバネティクスは身体に異物を入れる行為であり、等級Ⅱ以上は施術時とセッション開始時に「適応判定」を行う必要がある。

### 適応判定

**判定方法：** 魂で判定。達成値4以上で成功。

| 結果 | 効果 |
|---|---|
| スペシャル（6） | 完全適応。次のセッションまで判定不要 |
| 成功（4〜5） | 問題なし |
| 失敗（2〜3） | 拒絶反応：該当部位の能力修正が−1。次の適応判定まで持続 |
| ファンブル（1） | 重篤拒絶：侵食率+5%。サイバネティクスが1ラウンド機能停止。感情共鳴（渇望+2） |

### 修正値

| 条件 | 修正 |
|---|---|
| 蒼鉄機工製 | +1 |
| 等級Ⅲ | −1 |
| 蜃気楼工廠製 | −2 |
| 過積載（同部位2つ以上） | −1（1つにつき） |
| 侵食率50%以上 | −1 |

<a id="c7"></a>
## C-7　キャラクター作成への組込み

### 作成手順（追加ステップ）

キャラクター作成の⑧（装備選択）の後に以下を追加：

```
⑧-b サイバネティクスを選択（任意）
  → 改造等級を決める（Ⅰ/Ⅱ/Ⅲ）
  → 改造CPの範囲内で改造部位とサイバネティクスを選ぶ
  → 基礎侵食加算を初期侵食率に足す
  → 等級Ⅱ以上は適応判定を行う（キャラ作成時は自動成功）
```

### 改造CP

- 基本改造CP：**3CP**（全キャラ共通）
- 技術畑の背景：+2CP
- 技術屋（傭兵）：+2CP
- 実験覚醒型：+1CP（人体実験の経験）
- 機甲士クラス：+1CP

### 制約

- 等級ⅢはGMの許可が必要
- 蜃気楼工廠製の怪異核インプラントは無所属か鴉羽技研との繋がりが必要
- 改造を後から追加する場合、EP（経験値）とは別に「ダウンタイム」と「施術費」が必要
