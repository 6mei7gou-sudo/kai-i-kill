## H-1　PCのHP

### HP計算式

PCの最大HPは以下の合計で決まる。

```
最大HP = 基礎HP + 体ランクボーナス + 背景ボーナス + サイバネティクスボーナス
```

| 要素 | 値 |
|---|---|
| **基礎HP** | 10（全PC共通） |
| **体ランクボーナス** | D: +0 / C: +2 / B: +4 / A: +6 / S: +8 |
| **背景「鋼の肉体」** | +2 |
| **サイバネティクス**（人工心肺など） | 個別効果に記載 |

### ランク別の最大HP早見表

| 体ランク | 基本HP | 鋼の肉体 |
|---|---|---|
| D | 10 | 12 |
| C | 12 | 14 |
| B | 14 | 16 |
| A | 16 | 18 |
| S | 18 | 20 |

### HPに関する制約

- HPは0未満にならない
- 最大HPを超える回復はできない（超過分は切り捨て）
- **HPは非公開**にしてもよい（PC同士で把握していない演出も可能）
- 護衛のHPとPCのHPは別物。護衛には「耐久値」が設定される
- 高侵食率によるHP減少については [H-14](#h14) を参照

## H-2　ダメージの種類と処理

### 物理ダメージ

装備や素手による直接的な攻撃。護衛の攻撃もこれに該当する。

```
受けるダメージ = 護衛の攻撃値 − PCの防御修正
```

| PC防御修正の内訳 | 値 |
|---|---|
| 基礎防御 | 0（PCに素の防御値はない） |
| サイバネティクス「皮下装甲プレート」 | −1 |
| サイバネティクス「全身装甲皮膚」 | −2 |
| ギフト「恐怖の反射」 | −恐怖メーター蓄積点（1ターン） |
| 援護行動（COVER）成功 | ダメージ半減 |
| 回避専念（EVADE）成功 | ダメージ無効 |

### 魔法ダメージ

怪異のルールにより発生するダメージ。防御修正が適用されない場合が多い。

| 発生源 | ダメージ例 |
|---|---|
| 怪異のルール違反 | 固定ダメージ（等級による。四級で2点、二級で5点） |
| 護衛の魔法攻撃 | 護衛ステータスに記載 |
| 感染（INFECT）特性 | ダメージではなく共鳴メーター上昇 |

### 侵食ダメージ

侵食率の上昇に伴う間接的な身体変化。直接HPが減るわけではないが、以下の閾値を超えると能力にペナルティが発生する。詳細は [H-14](#h14) を参照。

| 侵食率 | HPへの影響 |
|---|---|
| 0〜25% | なし |
| 26〜50%（変容の兆し） | なし。ただし外見が変化し始める |
| 51〜75%（半化の影） | 最大HP −2 |
| 76〜99%（臨界） | 最大HP −5。毎ラウンド開始時にHP−1 |
| 100%（怪異化） | キャラクターロスト。PCはGM管理の怪異になる |

## H-3　状態異常

### 戦闘中の状態異常

| 状態 | 効果 | 回復 |
|---|---|---|
| **負傷（WOUND）** | 全判定−1 | 回復行動や応急処置で解除 |
| **重傷（CRITICAL）** | 全判定−2。移動速度半減 | セッション後の休息で回復 |
| **気絶（DOWN）** | 行動不能。攻撃対象にならない | 味方の援護行動（体判定成功）で1HP回復して復帰 |
| **混乱（CONFUSE）** | 次のターンの行動をGMが決定 | ラウンド終了で自動回復 |
| **恐慌（PANIC）** | 攻撃行動不可。回避専念か浄化のみ | 魂判定成功で解除 |
| **呪縛（BIND）** | 移動不可。回避不可 | 体判定（達成値5以上）で解除 |
| **暴走（BERSERK）** | GMが行動を決定。攻撃力+2だが味方も対象になり得る | ラウンド終了で自動回復。侵食率51%以上のPC専用 |

### 状態異常の発生タイミング

| トリガー | 状態異常 |
|---|---|
| HPが最大HPの半分以下 | 負傷（WOUND） |
| HPが3以下 | 重傷（CRITICAL） |
| HP0 | 気絶（DOWN）。生還の意地ギフトがあれば1HPで留まる |
| 護衛の「混乱（CONFUSE）」特性 | 混乱 |
| 恐怖メーター臨界（10点） | 恐慌（PANIC） |
| 一部の護衛特性・怪異ルール | 呪縛（BIND） |
| 高侵食PCの戦闘中ファンブル（[H-14](#h14)参照） | 暴走（BERSERK） |

## H-4　回復手段

### 戦闘中

| 手段 | 条件 | 回復量 | 備考 |
|---|---|---|---|
| **浄化行動（CLEANSE）** | 魂判定成功 | HP回復なし | 共鳴メーター−2。直接的なHP回復ではない |
| **自己修復魔導具**（装備オプション） | 自分のターンに発動 | 魂ランク数HP回復 | 1戦闘2回まで |
| **味方の援護行動** | 体判定成功 | 気絶者を1HP回復 | 気絶からの復帰用 |
| **人工心肺【強化型】**（サイバネティクス） | 常時 | 最大HP+3 | 回復ではなく底上げ |
| **信念ポイント消費** | 1点消費 | HP 3点回復 | いつでも使用可。宣言だけでよい |
| **魔法言語による回復行使** | 術判定成功 | 術ランク数HP回復 | 怪異誘発の可能性あり |

### 戦闘外（調査フェイズ中）

| 手段 | 条件 | 回復量 | 備考 |
|---|---|---|---|
| **休息** | 安全な場所で時間経過 | 最大HPの半分回復 | 1調査区切りにつき1回 |
| **食事** | 食料の確保 | HP 2点回復 | 定食屋や弁当。RP重視 |
| **応急処置** | 識判定成功 | HP 3点回復+負傷解除 | 医療キット必要 |
| **信念ポイント消費** | 1点消費 | HP 3点回復 | 戦闘中と同じ |

### セッション間（ダウンタイム）

| 手段 | 回復量 | 備考 |
|---|---|---|
| **完全休息** | HP全回復・負傷/重傷解除 | シナリオ間の休息期間 |
| **メーカー施術** | HP全回復+装備修理 | 費用が発生 |
| **蜃気楼工廠施術** | HP全回復+侵食率−5% | 極めて高額。副作用の可能性 |

## H-5　戦闘不能と死亡

### 戦闘不能（HP0）

HP0になったPCは**気絶（DOWN）**状態になる。

- 行動不能。一切のアクションが取れない
- 攻撃対象にはならない（護衛の攻撃は残った討伐者に向かう）
- 味方の援護行動で復帰可能（1HPで復帰）
- ギフト「生還の意地」があれば魂判定成功で1HPを残して気絶回避

### 死亡条件

PCの死亡は以下の条件で発生する。極めて重い結果なので、GMは事前に警告を出すことが推奨される。

| 条件 | 内容 |
|---|---|
| **HP0の状態で追加ダメージ** | 気絶中にさらにダメージを受けた場合、**死亡判定**を行う |
| **侵食率100%** | 怪異化してPCロスト。死亡ではないが、PLの手から離れる |
| **怪異のルールによる即死条件** | 一部の二級・一級怪異は「ルール違反3回で即死」などの条件を持つ |

### 死亡判定

HP0の状態でダメージを受けた場合、魂で判定する。

| 結果 | 内容 |
|---|---|
| スペシャル（6） | 奇跡の復活。HP1で復帰。全共鳴メーター+2 |
| 成功（4〜5） | まだ死んでいない。次の追加ダメージで自動死亡 |
| 失敗（2〜3） | 死亡。PLは死亡描写を行う権利を持つ |
| ファンブル（1） | 死亡。かつ怪異化の兆候（侵食率+30%の遺体が残る。シナリオ的な影響あり） |

## H-6　ダウンタイムと長期回復

シナリオ間の休息期間。PLはダウンタイムに以下の行動から**2つ**を選べる。

| 行動 | 効果 |
|---|---|
| **休息** | HP全回復。負傷・重傷解除 |
| **訓練** | EP（経験値）+1 |
| **装備整備** | 破損装備の修復。メーカー修理の場合はダウンタイム消費 |
| **サイバネティクス施術** | 新しいサイバネティクスの追加（費用・EP消費あり） |
| **情報収集** | 次のシナリオに関するヒントを1つ入手 |
| **人脈構築** | NPCとの関係を1段階深める |
| **闇市場アクセス** | 非正規の装備・素材・情報を入手（リスクあり） |

## H-7　具体的な戦闘例（基本）

### 状況設定

- PC「アキラ」：体B（HP14）、武装型装備（武器修正+2）、ギフト「闘争本能」
- PC「ユウキ」：術B・視野C（HP10）、半装身型装備（武器修正+1）
- 護衛A：HP6 / 行動値2 / 特性：干渉（BLOCK）
- 護衛B：HP4 / 行動値3 / 特性：感染（INFECT）
- 核：討伐クロック8

### ラウンド1

**行動宣言フェイズ：**
- アキラ → 護衛Aに攻撃（ATTACK）
- ユウキ → 護衛Bに魔法攻撃（SPELL）

**解決（疾ランク順）：**

アキラの攻撃（体B = 3d6）：
```
出目：5, 3, 4
達成値：5（成功） → 共鳴ダイス：4（怒りメーター+1）
ダメージ = 2 + 武器修正2 + 闘争本能1（怒り3以上）= 5
護衛Aに5ダメージ → 護衛AのHP: 6 → 1
まだ耐久値を超えていないので連鎖ダメージなし
```

ユウキの魔法攻撃（術B = 3d6）：
```
出目：6, 2, 4
達成値：6（スペシャル！）→ 共鳴ダイス：4（浄化+1）
ダメージ = 3 + 武器修正1 + スペシャル追加1 = 5
護衛Bに5ダメージ → 護衛BのHP: 4 → 0
耐久値4以上のダメージ → 連鎖ダメージ発動！
連鎖：4ダメージが核に伝播（スペシャルなので×1.5 = 6ダメージ）
核のクロック：8 → 2
```

**護衛の行動：**
```
護衛A（HP1）：アキラに攻撃。行動値2。
アキラのHP: 14 → 12（2ダメージ）
護衛B：行動不能（このラウンド終了まで）
ただし感染（INFECT）特性が発動 → ユウキの恐怖メーター+1
```

### ラウンド2

アキラが護衛Aに追い打ち。護衛A残りHP1のため、成功すれば連鎖で核に残り6ダメージ中一部を叩き込める。

制限ラウンド内にクロック0到達で討伐成功。

<a id="h8"></a>
## H-8　戦闘シーケンス詳細手順

> *「段取りを間違えた奴から死ぬ。手順を体に叩き込め。」*
> *—— 特務班 教導官*

核護衛戦の1ラウンドは5つのフェイズで構成される。各フェイズの処理を飛ばさず順番に解決すること。

### フェイズ一覧

```
① ラウンド開始処理
② 行動宣言フェイズ
③ 解決フェイズ
④ 護衛行動フェイズ
⑤ ラウンド終了処理
```

### ① ラウンド開始処理

ラウンドの冒頭で以下を処理する。PLの判断は不要。

| 処理 | 内容 |
|---|---|
| 再生チェック | 「再生（REGEN）」特性を持つ護衛がHP+2回復 |
| 侵食ダメージ | 侵食率76%以上のPCはHP−1 |
| 状態異常の自動回復 | 「混乱（CONFUSE）」「暴走（BERSERK）」は前ラウンドで付与されたものが解除 |
| ラウンドカウント | 現在のラウンド数を宣言。制限ラウンドの残りを全員に伝える |
| 環境効果の適用 | [H-9](#h9) の環境ルールがあれば処理 |

### ② 行動宣言フェイズ

**全PLが同時に行動を宣言する。** 宣言は公開でもよいし、紙に書いて同時に見せてもよい。

- 1人1行動が基本（「加速」ギフト使用時は2行動）
- 行動の種類は以下の8つから選択

| 行動 | 内容 | 使用能力値 |
|---|---|---|
| **攻撃（ATTACK）** | 護衛または核へのダメージ行動 | 体/疾 |
| **魔法攻撃（SPELL）** | 魔法を使った攻撃または支援 | 術 |
| **援護（COVER）** | 別のPCへの攻撃を肩代わり | 体 |
| **解明（DECODE）** | 戦闘中に解明鍵を進める | 識/視野 |
| **干渉（DISTURB）** | 護衛の特性を一時的に無効化 | 判 |
| **回避専念（EVADE）** | 攻撃不可だがダメージ無効 | 疾 |
| **浄化（CLEANSE）** | 共鳴メーター1種を−2点 | 魂 |
| **連携（SYNC）** | 他のPCとの連携行動。[H-12](#h12) 参照 | 行動による |

### ③ 解決フェイズ

**疾ランクが高いPC → 低いPC** の順で行動を解決する。同ランクはPL間で順番を決める。

各PCの行動につき：
1. 使用能力値でダイスを振る
2. 達成値ダイスと共鳴ダイスを選択
3. 達成値+修正で成功/失敗を判定
4. 成功ならダメージ計算（攻撃の場合）
5. 共鳴メーターを上昇させる
6. 連鎖ダメージの判定（護衛の耐久値以上のダメージ時）

**ファンブル（出目1）時は [H-11](#h11) の戦闘ファンブル表を参照。**

### ④ 護衛行動フェイズ

全PCの解決後、護衛が行動値の高い順に行動する。

| 護衛の行動 | 効果 |
|---|---|
| **攻撃** | 任意のPCに行動値分のダメージ |
| **妨害** | 任意のPCの次の判定に−1修正 |
| **庇い** | 核への攻撃を自分で受ける（1ラウンド1回） |
| **強化** | 別の護衛のHPを3回復 |

護衛の行動選択はGMが状況に応じて決定する。原則として以下の優先度に従う：

1. 核が直接攻撃を受けている → 庇い
2. 最もダメージを与えてきたPCがいる → そのPCに攻撃
3. 味方護衛が瀕死 → 強化
4. 上記いずれでもない → 最もHPの低いPCに攻撃

### ⑤ ラウンド終了処理

| 処理 | 内容 |
|---|---|
| クロック更新 | 核への攻撃成功・連鎖ダメージ伝播によるクロック減少を確認 |
| 討伐成功判定 | クロック0なら討伐成功。戦闘終了 |
| 制限ラウンド判定 | 制限ラウンド到達なら制限到達処理（延長権があるか確認） |
| 状態異常チェック | HPが閾値を下回ったPCに負傷/重傷を付与 |
| 制限ラウンド残り1の共鳴 | 残り1ラウンドの場合、全PCの焦燥メーター+1（強制） |

<a id="h9"></a>
## H-9　環境と地形

> *「場所を選べ。怪異のホームグラウンドで殴り合うのはただの馬鹿だ。」*
> *—— 白鷺調査社 所長*

戦闘が行われる空間によって、判定や戦闘にルール修正が発生する。

### 戦場の分類

| 戦場 | 概要 | 主な修正 |
|---|---|---|
| **現実空間（都市部）** | 人目のある一般的な空間 | 全装身型使用で都市伝説化リスク。一般人の避難が必要 |
| **現実空間（僻地）** | 人気のない場所 | 修正なし。標準的な戦場 |
| **怪異の領域** | 怪異のルールが空間を支配している場所 | 護衛の行動値+1。全PCの恐怖メーターがラウンド開始時+1 |
| **禁足地** | 公的に封鎖された怪異定着地 | 封印クロック方式。環境そのものが護衛として機能し得る |

### 環境修正

| 環境条件 | 修正 |
|---|---|
| **暗所**（完全な闇） | 視野・疾の判定−1。サーマルアイ持ちは免除 |
| **閉所**（狭い空間） | 全装身型・搭乗型の武器修正−1。武装型は影響なし |
| **高所**（落下リスク） | 疾判定にファンブルが出たら1d6の落下ダメージ |
| **水場**（水中・浸水区域） | 体判定−1。電気系装備が機能停止（NGT言語使用不可） |
| **電磁妨害**（ジャミング） | 独立型装備が行動不能。NGT魔法が使用不可 |
| **聖域**（神社・祠の近傍） | 浄化行動（CLEANSE）の効果が2倍（−4点）。古い怪異の護衛HP−2 |

### 怪異の領域の特殊ルール

怪異の領域内では、怪異のルールが強化される。

- ルール違反ダメージが1.5倍（端数切り上げ）
- 護衛が倒されたラウンドの終了時、怪異の領域の効果でGMは1d6を振る。4以上で倒された護衛が**HP半分で復活**する（1戦闘で各護衛1回まで）
- 怪異の領域から脱出するには、全PCが疾判定に成功する必要がある（[H-10](#h10) の撤退ルール参照）

## H-10　撤退と逃走

> *「逃げることは敗北じゃない。次に殺すための準備だ。」*
> *—— はぐれ狼の不文律*

戦闘から撤退することは可能だが、代償が伴う。

### 撤退の種類

| 種類 | 条件 | 判定 |
|---|---|---|
| **全体撤退** | 生存PC全員が合意 | 各PCが疾で判定。全員成功で安全離脱 |
| **個別逃走** | 1人だけ離脱 | 疾で判定。失敗すると護衛の追撃を受ける |
| **緊急離脱** | HP3以下のPCが逃走 | 疾で判定。成功で即座に戦場離脱。失敗しても追撃ダメージ半減 |

### 撤退の代償

撤退は「討伐を諦める」行為であり、以下の代償が生じる。

| 代償 | 内容 |
|---|---|
| **討伐クロック巻き戻し** | 撤退時点のクロック値に+2される（怪異が回復する） |
| **怪異のルール悪化** | 再戦時、ルール違反ダメージが+1点される |
| **信念消費** | 撤退したPC全員の信念ポイント−1 |
| **共鳴効果** | 全PCの哀愁メーター+1 |

### 撤退判定の詳細

**全体撤退：**

1. 行動宣言フェイズで全PCが「撤退」を宣言
2. 各PCが疾で判定（達成値4以上で成功）
3. 全員成功 → 安全離脱。護衛の追撃なし
4. 1人でも失敗 → 失敗したPCが護衛の追撃を受ける（行動値の高い護衛1体から攻撃）。その後離脱

**個別逃走：**

1. 行動宣言で「逃走」を宣言（他のPCは戦闘継続可能）
2. 疾で判定
3. 成功 → 離脱。残ったPCだけで戦闘続行
4. 失敗 → 護衛から追撃（行動値分のダメージ）を受けた上で離脱

### 再突入

撤退後に再び戦闘に入る場合の制約。

- 次のラウンドに再突入可能（ただし行動宣言はできない。再突入ラウンドは護衛の行動のみ受ける）
- 再突入時、制限ラウンドは撤退時から**カウント継続**（リセットされない）
- 怪異の領域の場合、再突入時に全PCの恐怖メーター+2

## H-11　戦闘ファンブル表

> *「最悪のタイミングで最悪のことが起きる。それが戦場だ。」*
> *—— 古怪班 副班長の遺言*

戦闘中に達成値ダイスでファンブル（出目1）が出た場合、**通常のファンブル処理に加えて**以下の表を使用する。GMが1d6を振って結果を決定。

| 出目 | 事象 | 効果 |
|---|---|---|
| **1** | **武器暴発** | 使用中の装備の改造1つが「破損状態」に（[H-13](#h13)参照）。改造がなければ武器修正が次ラウンドまで−1 |
| **2** | **同士打ち** | 最も近い味方PC1人に1ダメージ。対象がいない場合は自分に1ダメージ |
| **3** | **隙の発生** | 次のラウンド終了まで、このPCへの護衛の攻撃ダメージ+1 |
| **4** | **怪異の干渉** | 恐怖メーター+2（強制）。怪異のルールが一瞬PCの動きを縛った |
| **5** | **共鳴暴走** | ランダムに共鳴メーター1種を決定し、+3上昇。1d6で決定（1=恐怖、2=怒り、3=哀愁、4=焦燥、5=渇望、6=浄化） |
| **6** | **護衛強化** | HP最大の護衛1体がHP+3回復。全護衛が行動不能の場合はクロック+1 |

### スペシャル＆ファンブルの同時処理

ダイスプールに6と1の両方が含まれている場合、PLが達成値ダイスとして6を選べばスペシャルになる。1を選べばファンブルだ。ただし**共鳴ダイスとして1を選んだ場合も恐怖メーター+2は発生する**。

<a id="h12"></a>
## H-12　連携アクション

> *「一人で勝てる相手なら、最初から苦労してない。」*
> *—— 小さな群れのクラン長*

2人以上のPCが協力して行う特殊行動。通常の行動より強力だが、コストが掛かる。

### 連携の条件

- 参加する全PCが行動宣言で「連携（SYNC）」を宣言すること
- **信念ポイントを1点消費**すること（連携を発動するPC1人が消費）
- 参加PCは全員、同じラウンドに他の行動を取れない

### 連携アクション一覧

#### 同時攻撃（SYNC ATTACK）

2PCが同じ護衛を同時に攻撃する。

- 両者がそれぞれ攻撃判定を行う
- **両方成功した場合**：ダメージを合算して護衛に適用。連鎖ダメージ判定は合算値で行う
- 片方のみ成功：成功した側のダメージのみ適用（通常攻撃と同じ）
- 両方失敗：信念を消費した上で攻撃失敗

> **例：** アキラ（ダメージ5）とユウキ（ダメージ3）が同時攻撃。合算8ダメージを耐久値6の護衛に与える。連鎖条件を満たし、核に6ダメージ伝播。

#### 囮と突破（DECOY & STRIKE）

1PCが護衛の注意を引き、もう1PCが核への接近を試みる。

- 囮役：体で判定。成功すれば護衛の攻撃対象を1ラウンド自分に固定
- 突破役：疾で判定。成功すれば核への直接攻撃に+2修正（解明完了後のみ有効）
- 囮役は護衛全体から攻撃を受ける（行動値の合算分のダメージ）

#### 浄化支援（PURGE ASSIST）

1PCが浄化行動を行い、もう1PCがそれを護る。

- 浄化役：魂で判定（通常の浄化行動）。判定に+1修正
- 護衛役：体で判定。成功すれば浄化中のPCへのダメージを全て肩代わり
- 浄化の効果が通常−2から**−3**に強化

#### 情報共有（INTEL SHARE）

戦闘中に情報を共有し、全体の判定精度を上げる。

- 発動条件：参加PCの1人が識ランクC以上
- 識で判定。成功すれば**次のラウンド中**、全PCの判定に+1修正
- 失敗しても信念消費のみ。ペナルティなし

<a id="h13"></a>
## H-13　戦闘中の装備ダメージ

> *「武器が壊れた？ なら拳で殴れ。壊れない武器は己の体だけだ。」*
> *—— 路地裏の犬、暗黙のルール*

### 装備が破損する条件

| 条件 | 効果 |
|---|---|
| 戦闘ファンブル表の「武器暴発」（[H-11](#h11)） | 改造1つが破損状態 |
| 行動値5以上の護衛から攻撃を受けた場合 | GM判断で装備破損判定（1d6で1が出たら破損） |
| 怪異のルールによる装備干渉 | 怪異データに記載された効果に従う |
| 怪異核素材装備の侵食ロールでファンブル | 装備が暴走。侵食率+5%かつ装備の武器修正が1ラウンド+2（制御不能な出力増大） |

### 破損状態の影響

| 状態 | 武器修正への影響 | 改造の効果 |
|---|---|---|
| **正常** | 通常 | 通常 |
| **破損** | −1 | 効果半減（端数切り捨て） |
| **大破** | −2 | 使用不可 |

- 1戦闘中に同じ装備が2回破損すると**大破**に変わる
- 大破した装備は戦闘中に修復できない

### 戦闘中の応急修理

行動1消費で応急修理を試みることができる。

- **判定：** 識で判定（達成値4以上で成功）
- **成功：** 破損状態1つを解除（正常に戻る）
- **失敗：** 修理失敗。行動を消費しただけで終わる
- **修正値：** 技術班所属なら+2。技術屋（傭兵）なら+1

## H-14　高侵食時の戦闘特殊ルール

> *「あいつの目を見ろ。もう半分はこっち側じゃない。」*
> *—— 煙管組の古参*

侵食率が高い討伐者は、戦闘において通常のPCとは異なるルール効果を受ける。人間としての力が弱まる代わりに、怪異に近い知覚や耐性を得る。

### 侵食段階別の戦闘効果

#### 変容の兆し（侵食率 26〜50%）

| 効果 | 内容 |
|---|---|
| **怪異感知** | 護衛の特性を1つ、戦闘開始時に自動で把握できる |
| **HPへの影響** | なし |
| **代償** | 外見が少し変わっている。NPCが違和感を感じることがある |

#### 半化の影（侵食率 51〜75%）

| 効果 | 内容 |
|---|---|
| **最大HP** | −2 |
| **怪異感知強化** | 護衛の特性を全て把握。護衛のHPも可視化される |
| **ルール部分耐性** | 怪異のルール違反ダメージを1点軽減 |
| **暴走リスク** | 戦闘中のファンブル時、追加で魂判定。失敗すると「暴走（BERSERK）」状態に（[H-3](#h3)参照） |

#### 臨界（侵食率 76〜99%）

| 効果 | 内容 |
|---|---|
| **最大HP** | −5 |
| **毎ラウンドダメージ** | ラウンド開始時にHP−1 |
| **怪異の知覚** | 核のクロック値を常時感知。護衛の次の行動を予測（GMが護衛の行動を事前に公開） |
| **ルール耐性** | 怪異のルール違反ダメージが半減 |
| **GMの「引き」** | 護衛を倒した直後に魂判定。失敗すると次のラウンドの行動をGMが決定する（暴走状態と同等） |
| **暴走リスク強化** | ファンブル時は魂判定なしで自動的に暴走状態に |

### 高侵食PCと連携

- 高侵食PC（51%以上）は**連携アクションの信念コストが0**になる（人間としての枠を超えた直感で連携が取れる）
- ただし連携相手の恐怖メーター+1（高侵食PCの異質さに触れたため）

## H-15　拡充戦闘例

### 戦闘例1：3人パーティ vs 三級怪異 — 連携と撤退

#### 状況設定

- PC「ミナト」：体B・疾C（HP14）、全装身型（武器修正+3）、機甲士クラス
- PC「レイ」：術B・魂C（HP10）、武装型（武器修正+2）、祓士クラス
- PC「カイ」：識B・視野C（HP12）、独立型（武器修正+1）、解明師クラス、侵食率30%
- 怪異：三級・乙種「呟き女」
- 護衛A《沈黙の壁》：HP8 / 行動値4 / 特性：干渉（BLOCK）、反射（REFLECT）
- 護衛B《拡散する声》：HP5 / 行動値3 / 特性：感染（INFECT）
- 核：討伐クロック8 / 制限ラウンド3（+1延長可）

#### ラウンド1

**① ラウンド開始処理：** 初回のため処理なし。環境「怪異の領域」→ 護衛の行動値+1（護衛A=5、護衛B=4）

**② 行動宣言：**
- ミナト＆レイ → 連携「同時攻撃（SYNC ATTACK）」で護衛Aを狙う。レイが信念1消費
- カイ → 解明（DECODE）で戦闘中の解明鍵を進める

**③ 解決フェイズ：**

ミナトの攻撃（体B = 3d6）：
```
出目：4, 5, 2
達成値：5（成功） → 共鳴ダイス：4（怒り+1）
ダメージ = 2 + 武器修正3 = 5
```

レイの攻撃（術B = 3d6）：
```
出目：3, 4, 6
達成値：6（スペシャル！）→ 共鳴ダイス：4（怒り+2）
ダメージ = 3 + 武器修正2 + スペシャル1 = 6
```

同時攻撃合算：5 + 6 = **11ダメージ** → 護衛A（HP8）に適用
```
護衛AのHP: 8 → 0（行動不能）
耐久値8以上のダメージ → 連鎖ダメージ発動
伝播ダメージ = 8（スペシャル含みなので×1.5 = 12）
核のクロック: 8 → 0！ → 制限ラウンド1で討伐成功！
```

ただし護衛Aは反射（REFLECT）特性を持つ。**ダメージの半分（5点）がミナトに跳ね返る。**

```
ミナトのHP: 14 → 9
状態チェック：まだ半分以上あるので負傷なし
ミナトの恐怖メーター+1（ダメージを受けた）
```

カイの解明行動（識B = 3d6）：
```
出目：5, 3, 1
達成値：5（成功）→ 共鳴ダイス：1 → 恐怖メーター+2（ファンブルの共鳴ダイス）
解明鍵④（ルールの全容）を取得
```

怪異の領域効果で護衛復活判定：GM 1d6 → 出目3 → 復活なし。

#### 結果

連携アクションの火力が突破力を生んだ。機甲士の連鎖条件緩和（耐久値−2以上で発動）がなくても圧倒できたケースだが、反射特性によるカウンターダメージには注意が必要。

### 戦闘例2：ソロ戦闘 — 高侵食PCの限界戦闘

#### 状況設定

- PC「ジン」：体A・魂B（HP11 ※侵食で最大HP−5）、侵食率82%（臨界段階）、武装型（武器修正+4、怪異核素材組込み）
- 怪異：四級・甲種「反転鏡」
- 護衛《映し身》：HP6 / 行動値3 / 特性：反射（REFLECT）
- 核：討伐クロック6 / 制限ラウンド3

ジンは侵食率82%のため、以下の特殊ルールが適用：
- 毎ラウンドHP−1
- 護衛の次の行動を予測（GMが事前公開）
- 怪異のルール違反ダメージ半減
- ファンブル=自動暴走

#### ラウンド1

**① 開始処理：** 侵食ダメージ HP11→10

**② 行動宣言：** ジン → 護衛に攻撃（ATTACK）

**③ 解決（体A = 4d6）：**
```
出目：6, 4, 3, 2
達成値：6（スペシャル！）→ 共鳴ダイス：4（怒り+2）
ダメージ = 3 + 武器修正4 + スペシャル1 = 8
護衛HP: 6 → 0 → 連鎖ダメージ発動
伝播 = 6（スペシャル ×1.5 = 9）
核のクロック: 6 → 0 → 討伐成功！
```

護衛の反射でダメージ半分（4点）がジンに返る：
```
ジンのHP: 10 → 6
```

**「引き」判定：** 護衛を倒したので魂判定（魂B = 3d6）
```
出目：5, 4, 2
達成値：5（成功）→ 暴走回避。ジンはまだ人間だ。
```

#### 結果

体Aランクのダイスプールとスペシャルによるワンショットキル。高侵食PCの持つ火力と引き換えのリスクが克明に出たケース。ファンブルが出ていれば暴走→討伐失敗の可能性もあった。

### 戦闘例3：特級怪異の封印クロック戦闘（GM向け進行例）

#### 状況設定

- 怪異：特級・丙種「白銀の廃線《タクトライン》」
- 封印クロック方式：上限16
- PC4人パーティ（祓部2、傭兵1、無所属1）
- 護衛は「ルールを破った場合のみ出現」する特殊仕様

#### 進行の概要

**序盤（封印クロック 0〜5）：**
調査と封印符の設置がメイン。戦闘は発生しない。禁足地内部での探索と、路線図の位置特定が主な行動。識・視野の判定が中心。

**中盤（封印クロック 6〜12）：**
路線図への封印作業中にルールを破りかけるPCが出てくる（「次の駅を思い浮かべてしまう」）。ルール違反で護衛が出現。

護衛《銀の列車》：HP10 / 行動値5 / 特性：連結（CHAIN）
護衛《時刻表の影》：HP6 / 行動値4 / 特性：干渉（BLOCK）

護衛を倒しても封印クロックは増えない。**封印行動（CLEANSE の封印版）** で魂判定に成功するとクロック+1〜+2。

**終盤（封印クロック 13〜16）：**
最後の路線図が最も深部にある。全PCの魂判定の連続成功が要求される。1人でも失敗すると護衛が再出現。

封印クロック16到達で「管理封印状態」→ シナリオクリア。

#### GM向けポイント

- 戦闘力で押し切れない設計が特級怪異の本質。PLには「戦わずに封印する」発想の切り替えが必要
- 護衛は討伐するより**無力化して時間を稼ぐ**方が効率的
- 制限ラウンドは存在しない代わりに、ルール違反の蓄積が「実質的なタイムリミット」として機能する
