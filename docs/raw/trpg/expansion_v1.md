## EX-0-2　解禁レベルと判断基準

本書のデータは3段階の解禁レベルに分かれる。キャンペーンの進行に応じてGMがPLへ段階的に開放する。

| Lv | 名称 | 解禁タイミング | 対応章 | GMの判断基準 |
|---|---|---|---|---|
| **Lv.1** | 闇市場 | 序盤〜中盤 | EX-1, EX-2, EX-3 | PLが闇市場に接触した、または出自不明の装備を入手した時点 |
| **Lv.2** | 真相 | 中盤〜後半 | EX-4, EX-5, EX-6 | PLが教団の存在を認知した、または怪異核素材の侵食が進行した時点 |
| **Lv.3** | 禁域 | 終盤〜特別シナリオ | EX-7, EX-8 | PLが二級以上の怪異と対峙する、または特別シナリオで解禁が必要な時点 |

### 解禁の原則

- **早すぎる解禁は世界観を壊す。** 教団の存在を序盤で明かすと、三勢力の緊張が台無しになる
- **遅すぎる解禁はPLの選択肢を狭める。** 闇市場装備は中盤までに開放しないとビルドの自由度が下がる
- **段階をまたぐ解禁は慎重に。** Lv.1を飛ばしてLv.2を開放することは避ける
- **EX-9 はどのレベルでも参照可能。** ビルド例とシナリオフックはGMが適宜選択して使用する

| 規模感 | 不明。単独の組織か複数の製造者のブランド名か判明していない |

> 闇市場には「蜃気楼工廠」の刻印がある装備が流通している。粗雑な闇市場品ではなく、明らかに高度な技術と素養持ちの技術者が関与した製品だ。使用者への代償が設計に織り込まれているような構造をしており、「壊れた怪異核由来」と「生きた怪異核由来」の二種が明確に区別されている。祓部は製造者の特定を試みているが、拠点の場所・関係者の素性・製造工程の全てが謎のまま残っている。

## EX-1-4　GM裏設定メモ

> **⚠ GM限定データ**：このセクションはGM専用。PLには見せないこと。

### 蜃気楼工廠と教団の関係

- 蜃気楼工廠と教団の間に直接的な証拠はないが、以下の状況証拠がある
  - 蜃気楼製品の流通量が教団の活動と連動している
  - 一部の蜃気楼製品に教団の儀式で使われる紋様が刻まれている
  - 教団の幹部が蜃気楼製品を所持している目撃情報がある
- **GMの判断指針**：蜃気楼工廠が教団のフロント組織なのか、教団とは独立した第四の勢力なのかはGMが決定する。キャンペーンの方向性に合わせて設定すること

### 蜃気楼工廠と雷禽重工の関係

- 雷禽重工の特別研究部門が蜃気楼工廠の製品を研究用に入手している情報がある
- 両者の技術的な類似性を指摘する分析もあるが、確定情報はない

# EX-2　怪異核素材 装備・オプション

> *「この刀は斬った怪異の夢を見る。お前もいずれ見るようになる。」*
> *—— 骨董屋の傭兵*

**解禁レベル：Lv.1 闇市場**

---

## EX-2-1　概要

怪異核素材とは、怪異の核を素材として組み込んだ魔導具の総称だ。合法的な流通経路は存在しない。

### PL配布用導入文

> 怪異を討伐すると核が残る。通常は祓部が回収・封印するが、一部はそのまま闇市場に流れる。この核を素材として武器や装備に組み込んだものが「怪異核素材」だ。二種類ある——まだ「生きている」核と、すでに「壊れた」核だ。どちらも使用者に力を与える。ただし代償がある。

### 二種の違い

| | 生きた怪異核素材 | 壊れた怪異核素材 |
|---|---|---|
| 核の状態 | 怪異としての意志が残存 | 怪異としての意志は消失 |
| 主な効果 | 高い攻撃力・怪異の能力の部分的行使 | 感知能力の拡張・解明の優位 |
| 侵食の方向 | 使用者が怪異のルールに侵食される（怪異化） | 人間的な感情と価値観が薄れる（感情平板化） |
| 侵食速度 | 速い（+10%/使用） | 遅い（+5%/使用） |
| 外見変化 | 大きい（怪異に似てくる） | 小さい（気づきにくい） |
| 可逆性 | **不可逆** | **不可逆** |

# EX-3　怪異核素材 サイバネティクス

> *「肉を捨てるか、魂を捨てるか。どっちも捨てたくないなら最初から手を出すな。」*
> *—— 鴉羽技研 闇施術師*

**解禁レベル：Lv.1 闇市場**

## EX-3-1　定義

怪異核インプラントは、怪異の核を直接人体に埋め込むサイバネティクス処置だ。通常のサイバネティクス（cybernetics_v1.md参照）とは以下の点で異なる。

| | 通常のサイバネティクス | 怪異核インプラント |
|---|---|---|
| 素材 | 魔導機関（金属・結晶） | 怪異の核（生きた核 or 壊れた核） |
| 取り外し | **不可逆**（通常のサイバネも不可逆） | **不可逆**かつ核が身体と融合する |
| 侵食リスク | 等級に応じた基礎侵食加算 | 基礎侵食加算+**使用するたびに追加侵食** |
| 拒絶反応 | 魂判定で対処可能 | 魂判定に追加のペナルティ。完全適応は困難 |
| 性能 | 能力値ボーナス中心 | 怪異の能力の部分的獲得 |

### 装備との違い（再確認）

怪異核素材の「装備」（EX-2）と「インプラント」（EX-3）の最大の違いは**着脱可能性**だ。

- **装備**：手放すことができる。使わなければ侵食は進まない
- **インプラント**：身体の一部になる。手放すことは二度とできない。核が体内で「呼吸」し続ける

## EX-3-4　核依存と離脱不能性

怪異核インプラントを施術した討伐者は、核との間に「依存関係」が発生する。

### 核依存の段階

| 侵食率 | 依存段階 | 効果 |
|---|---|---|
| 施術直後〜25% | **共存期** | 核は静かに機能する。使用者はほぼ通常通り |
| 26〜50% | **馴化期** | 核の存在が当たり前になる。核を使わない判定に−1の衝動修正（魂判定で抵抗可） |
| 51〜75% | **癒着期** | 核が身体機能の一部を担い始める。核の機能を意図的に停止すると全判定−2 |
| 76〜99% | **融合期** | 核と使用者の境界が曖昧になる。核の除去はHP0+死亡判定が発生する危険行為 |

### 離脱の不可能性

- 怪異核インプラントは**除去できない**。手術で取り出そうとすると、核が癒着した組織ごと摘出することになり、等級IIIの改造を「巻き戻す」こと以上の身体損傷が発生する
- 核を「殺す」ことは理論上可能だが、殺した核はそのまま「壊れた怪異核」として体内に残る。侵食は止まらない
- **結論：一度施術したら、侵食率100%になるまで共に歩むしかない**

## EX-4-2　教団の構造

### 表の顔

合法的な宗教法人または福祉団体として活動している。「あなたの体験は本物だ」という言葉が、怪異の存在を確信しながら孤独を抱える一般人を引き寄せる最大の武器だ。

### 組織の層構造

|---|---|---|

| **実務幹部** | 教祖の最終目的を知らされていない管理職 | 部分的に知っている。担当領域ごとに分割されたセル構造 |
| **側近** | 教祖直属の4人 | 完全に知っている。各自の動機で教祖に従う |
| **教祖** | 1人 | 全てを知っている。怪異でも神でもない「第三の存在」への変容を目指す |

### 気づいた信者の行く末

一般信者の1%が断片的な違和感から教団の真実に気づくことがある。

| 選択肢 | 結果 |
|---|---|
| **留まる** | 実務幹部候補として取り込まれる。情報は与えられるが自由は失われる |
| **取り込まれる** | 教団の実験に自発的に参加する。実験覚醒型の候補になる |
| **離反する** | 教団を離れようとする。教団は「離反者」を放置しない |

## EX-4-3　怪異制御方法

教団だけが怪異を意図的に兵器化できる。その手段は三種ある。

| 手段 | 概要 | 使用者 | リスク |
|---|---|---|---|
| **儀式型** | 特定の手順・言葉・供物で怪異のルールに干渉する | 実務幹部レベル | 低〜中。手順を間違えると術者が怪異に呑まれる |
| **素養直接行使型** | 魔法を直接行使してルールを書き換える | 側近レベル | 中〜高。高い素養と侵食耐性が必要 |
| **媒介型** | 教団員自身が怪異の核の媒介になる | 教祖のみが許可 | 最大。媒介者の侵食率は急速に上昇し、最終的に怪異化する |

### 制御方法のゲーム的処理

| 手段 | GMの処理 |
|---|---|
| 儀式型 | 怪異のルールを1つ「書き換える」ことができる。PLが調査で得た情報が無効化される可能性がある |
| 素養直接行使型 | 護衛に「強化（ENHANCE）」特性を追加する。護衛のHPと行動値が+2される |
| 媒介型 | 怪異に「新しいルール」を1つ追加する。媒介者はNPCとして護衛の一部になる |

| 代償 | 効果 | 演出 |

| **慢性疼痛** | 毎セッション開始時にHP−1。戦闘外の体判定に−1 | 常時どこかが痛む。気温や湿度で悪化する |

| **代謝異常** | 食事による回復が無効。信念ポイントによる回復量が2点に低下 | 通常の食事を受け付けない。特殊な栄養剤が必要 |

## EX-5-4　教団からの逃亡と追跡

教団由来の実験覚醒型PCは「追跡状態」で開始する。

### 追跡ルール

| 項目 | 内容 |
|---|---|
| 追跡レベル | 1（低）〜5（最高）の5段階。初期値は2 |
| レベル上昇条件 | 教団関連のシナリオで目撃された、教団の怪異を討伐した、教団員と交戦した |
| レベル低下条件 | 3セッション連続で教団と接触しなかった場合、−1 |

### 追跡レベルの効果

| レベル | 効果 |
|---|---|
| **1（低）** | 教団はPCの現在位置を把握していない。シナリオへの介入なし |
| **2（警戒）** | セッション開始時にGMが1d6。5以上で教団の偵察員が出現（非戦闘） |
| **3（捜索）** | 調査フェイズで教団の妨害が1回発生（判定に−1修正の情報撹乱） |
| **4（追撃）** | セッション中に教団の戦闘員1〜2名が接触。戦闘が発生する可能性あり |
| **5（包囲）** | セッションのクライマックスで教団の側近級NPCが出現。討伐対象の怪異に加えて教団との同時対処を強いられる |

### ドラマルール：追跡者との対決

追跡レベルが4以上の場合、GMは「追跡者」を名前付きNPCとして設定することを推奨する。追跡者はPCの過去を知っており、教団内での立場に応じた情報をPCに提示する。

> **例：** 追跡者「灰原」はPCと同時期に実験を受けた被験者だ。教団を離れられなかった灰原はPCの回収を命じられている。だが灰原自身も教団に疑問を持ち始めている。

# EX-6　怪異核素材 侵食メカニズム

> *「あいつは壊れた核を使い続けた。痛みも怒りも消えていった。最後に残ったのは、怪異を視る目だけだ。」*
> *—— 白鷺調査社 所員の手記*

## EX-6-1　侵食原理

怪異核素材の侵食は、通常の異能使用による侵食（rules CH7）とは別のメカニズムで進行する。

| | 異能による侵食 | 生きた核による侵食 | 壊れた核による侵食 |
|---|---|---|---|
| 原因 | 使用者自身の変質 | 核に宿る怪異の意志が使用者に干渉 | 核の残留エネルギーが使用者を浸食 |
| 侵食方向 | 使用者の異能が怪異的性質を帯びる | 使用者が核の怪異に似ていく（怪異化） | 使用者の人間性が薄れていく（感情平板化） |
| 速度 | 行使結果に依存 | +10%/使用 | +5%/使用 |
| 外見変化 | 中程度 | 大きい（怪異に似る） | 小さい（気づきにくい） |
| 加算関係 | 独立 | 異能侵食と加算 | 異能侵食と加算 |

### 重要：侵食率は一本の値

異能の侵食も怪異核素材の侵食も、同じ「侵食率」に加算される。別々のトラックは存在しない。侵食率100%に達した時点でキャラクターは終了する。

### 侵食進行

| タイミング | 侵食率上昇 | 追加効果 |

| 戦闘中に生きた核装備でファンブル | +10%+追加10% | 核が「暴走」する。暴走（BERSERK）状態に |

### 怪異化の進行

生きた核による侵食が進むと、使用者は核に宿っていた怪異の特徴を帯びていく。

| 侵食率 | 変化 | ゲーム的効果 |
|---|---|---|
| 26〜50% | 核の怪異に関連する外見変化が始まる | 察判定+1。対人判定−1の可能性（GM判断） |

| 76〜99% | 核の怪異と「対話」できるようになる | 護衛の行動を予測可能。ただしGMの「引き」が毎シーン |
| 100% | 核の怪異と融合。PCロスト | GMが新しい怪異として管理する。核の怪異のルールを引き継ぐ |

### 核依存の発生

生きた核装備を3回以上使用した時点で「核依存」が発生する。

| 効果 | 内容 |
|---|---|
| 渇望の増幅 | 渇望メーターの上昇が常時+1 |
| 使用衝動 | 戦闘開始時に魂判定。失敗すると最初の行動で必ず生きた核装備を使用しなければならない |
| 離脱困難 | 装備を手放す場合、魂判定（達成値5以上）が必要。失敗すると手放せない |

怪異核素材による侵食は不可逆だ。以下を明文化する。

| 渇望メーター+3 | 施術自体が怪異核素材を使用するため |
| 副作用判定 | 魂で判定。失敗すると侵食率が下がる代わりに身体代償が1つ追加される |

### 環境制限（H-9 参照）

| 環境 | 全装身型への影響 |
|---|---|
| 閉所 | 武器修正−1。回避行動不可 |
| 水場 | 全判定−2。行動不能のリスク（魂判定で回避） |
| 高所 | 落下ダメージ2倍（重量のため） |

### 同調維持と戦闘中の再判定

同調判定は戦闘開始時に1回行うが、以下の条件で**再判定**が必要になる。

| 条件 | 再判定のタイミング |
|---|---|
| HPが最大HPの半分以下になった | 即座に魂判定 |
| 暴走状態から復帰した直後 | 次のラウンド開始時に魂判定 |
| 侵食率が新たな段階に入った | 即座に魂判定 |

## EX-9-5　シナリオフック

### Lv.1 闇市場フック

**「蜃気楼の刻印」**
闇市場で入手した装備に蜃気楼工廠の刻印があった。だがその装備が「持ち主を選ぶ」かのように異常な挙動を見せる。装備の出自を追うと、煙管組の道具番に辿り着く。

> 導入キー：闇市場アクセス → 蜃気楼製装備の入手 → 異常挙動の調査

**「壊れた核の声」**
PCの一人が壊れた怪異核素材の武器を使い続けている。ある日、武器から「声」が聞こえ始める。壊れたはずの核が「思い出」を語りかけてくる。声の内容は次のシナリオの怪異に関するヒントだ。

> 導入キー：壊れた核装備3回以上使用 → 声の発生 → 怪異の過去を調査

### Lv.2 真相フック

**「教団の贈り物」**
教団がPLの一人に「贈り物」を送ってくる。中身は生きた怪異核素材の武器だ。使えば圧倒的な力を手に入れられるが、教団の「好意」には必ず裏がある。

> 導入キー：教団との接触 → 贈り物の受領判断 → 教団の真意の調査

**「逃亡者の証言」**
教団から逃亡した実験覚醒型の討伐者がPLに助けを求める。教団の追跡者が迫っており、逃亡者は教団内部の情報と引き換えに保護を要求する。

> 導入キー：逃亡者との接触 → 教団の追跡者との戦闘 → 内部情報の精査

**「灰色の感情」**
壊れた怪異核素材を長期使用しているPCが、仲間との感情的なやり取りに違和感を覚え始める。感情平板化が進行している。治療法を探す旅に出るが、蜃気楼工廠の施術だけが唯一の可能性だ。

> 導入キー：侵食率51%以上で対人判定失敗 → 感情喪失の自覚 → 蜃気楼工廠の捜索

### Lv.3 禁域フック

**「雷帝の暴走」**
雷禽重工の試作超大型機【雷帝】がテスト中に暴走し、パイロットを取り込んだまま都市部に出現した。祓部と傭兵が共同作戦を組み、PLは戦闘用搭乗型に搭乗して対処する。

> 導入キー：雷禽の緊急依頼 → 戦闘用搭乗型の貸与 → 暴走機体との戦闘

**「教祖の儀式」**
教祖が「第三の存在」への変容を試みる儀式を開始した。儀式が完了すると教祖は怪異でも人間でもない存在になり、既存のルールでは対処できなくなる。PLは全装身型・戦闘用搭乗型を含む全戦力を投入してクライマックスに挑む。

> 導入キー：教団拠点の特定 → 儀式の進行阻止 → 教祖との最終戦

**「異形巨体の覚醒」**
禁足地に封印されていた特級怪異が教団の手で「異形巨体」に改造され、覚醒した。封印クロック方式では対処できないほどの脅威に対し、PLは戦闘用搭乗型で直接討伐を試みる。

> 導入キー：禁足地の封印崩壊 → 異形巨体の出現 → 総力戦
