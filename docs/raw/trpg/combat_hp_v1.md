## H-1　PCのHP

### HP計算式

PCの最大HPは以下の合計で決まる。

```
最大HP = 基礎HP + 体ランクボーナス + 背景ボーナス + サイバネティクスボーナス
```

| 要素 | 値 |
|---|---|
| **基礎HP** | 10（全PC共通） |
| **体ランクボーナス** | D: +0 / C: +2 / B: +4 / A: +6 / S: +8 |
| **背景「鋼の肉体」** | +2 |
| **サイバネティクス**（人工心肺など） | 個別効果に記載 |

### ランク別の最大HP早見表

| 体ランク | 基本HP | 鋼の肉体 |
|---|---|---|
| D | 10 | 12 |
| C | 12 | 14 |
| B | 14 | 16 |
| A | 16 | 18 |
| S | 18 | 20 |

### HPに関する制約

- HPは0未満にならない
- 最大HPを超える回復はできない（超過分は切り捨て）
- **HPは非公開**にしてもよい（PC同士で把握していない演出も可能）
- 護衛のHPとPCのHPは別物。護衛には「耐久値」が設定される

## H-2　ダメージの種類と処理

### 物理ダメージ

装備や素手による直接的な攻撃。護衛の攻撃もこれに該当する。

```
受けるダメージ = 護衛の攻撃値 − PCの防御修正
```

| PC防御修正の内訳 | 値 |
|---|---|
| 基礎防御 | 0（PCに素の防御値はない） |
| サイバネティクス「皮下装甲プレート」 | −1 |
| サイバネティクス「全身装甲皮膚」 | −2 |
| ギフト「恐怖の反射」 | −恐怖メーター蓄積点（1ターン） |
| 援護行動（COVER）成功 | ダメージ半減 |
| 回避専念（EVADE）成功 | ダメージ無効 |

### 魔法ダメージ

怪異のルールにより発生するダメージ。防御修正が適用されない場合が多い。

| 発生源 | ダメージ例 |
|---|---|
| 怪異のルール違反 | 固定ダメージ（等級による。四級で2点、二級で5点） |
| 護衛の魔法攻撃 | 護衛ステータスに記載 |
| 感染（INFECT）特性 | ダメージではなく共鳴メーター上昇 |

### 侵食ダメージ

侵食率の上昇に伴う間接的な身体変化。直接HPが減るわけではないが、以下の閾値を超えると能力にペナルティが発生する。

| 侵食率 | HPへの影響 |
|---|---|
| 0〜25% | なし |
| 26〜50%（変容の兆し） | なし。ただし外見が変化し始める |
| 51〜75%（半化の影） | 最大HP −2 |
| 76〜99%（臨界） | 最大HP −5。毎ラウンド開始時にHP−1 |
| 100%（怪異化） | キャラクターロスト。PCはGM管理の怪異になる |

## H-3　状態異常

### 戦闘中の状態異常

| 状態 | 効果 | 回復 |
|---|---|---|
| **負傷（WOUND）** | 全判定−1 | 回復行動や応急処置で解除 |
| **重傷（CRITICAL）** | 全判定−2。移動速度半減 | セッション後の休息で回復 |
| **気絶（DOWN）** | 行動不能。攻撃対象にならない | 味方の援護行動（体判定成功）で1HP回復して復帰 |
| **混乱（CONFUSE）** | 次のターンの行動をGMが決定 | ラウンド終了で自動回復 |
| **恐慌（PANIC）** | 攻撃行動不可。回避専念か浄化のみ | 魂判定成功で解除 |
| **呪縛（BIND）** | 移動不可。回避不可 | 体判定（達成値5以上）で解除 |

### 状態異常の発生タイミング

| トリガー | 状態異常 |
|---|---|
| HPが最大HPの半分以下 | 負傷（WOUND） |
| HPが3以下 | 重傷（CRITICAL） |
| HP0 | 気絶（DOWN）。生還の意地ギフトがあれば1HPで留まる |
| 護衛の「混乱（CONFUSE）」特性 | 混乱 |
| 恐怖メーター臨界（10点） | 恐慌（PANIC） |
| 一部の護衛特性・怪異ルール | 呪縛（BIND） |

<a id="h4"></a>
## H-4　回復手段

### 戦闘中

| 手段 | 条件 | 回復量 | 備考 |
|---|---|---|---|
| **浄化行動（CLEANSE）** | 魂判定成功 | HP回復なし | 共鳴メーター−2。直接的なHP回復ではない |
| **自己修復魔導具**（装備オプション） | 自分のターンに発動 | 魂ランク数HP回復 | 1戦闘2回まで |
| **味方の援護行動** | 体判定成功 | 気絶者を1HP回復 | 気絶からの復帰用 |
| **人工心肺【強化型】**（サイバネティクス） | 常時 | 最大HP+3 | 回復ではなく底上げ |
| **信念ポイント消費** | 1点消費 | HP 3点回復 | いつでも使用可。宣言だけでよい |
| **魔法言語による回復行使** | 術判定成功 | 術ランク数HP回復 | 怪異誘発の可能性あり |

### 戦闘外（調査フェイズ中）

| 手段 | 条件 | 回復量 | 備考 |
|---|---|---|---|
| **休息** | 安全な場所で時間経過 | 最大HPの半分回復 | 1調査区切りにつき1回 |
| **食事** | 食料の確保 | HP 2点回復 | 定食屋や弁当。RP（ロールプレイ）重視 |
| **応急処置** | 識判定成功 | HP 3点回復+負傷解除 | 医療キット必要 |
| **信念ポイント消費** | 1点消費 | HP 3点回復 | 戦闘中と同じ |

### セッション間（ダウンタイム）

| 手段 | 回復量 | 備考 |
|---|---|---|
| **完全休息** | HP全回復・負傷/重傷解除 | シナリオ間の休息期間 |
| **メーカー施術** | HP全回復+装備修理 | 費用が発生 |
| **蜃気楼工廠施術** | HP全回復+侵食率−5% | 極めて高額。副作用の可能性 |

## H-5　戦闘不能と死亡

### 戦闘不能（HP0）

HP0になったPCは**気絶（DOWN）**状態になる。

- 行動不能。一切のアクションが取れない
- 攻撃対象にはならない（護衛の攻撃は残った討伐者に向かう）
- 味方の援護行動で復帰可能（1HPで復帰）
- ギフト「生還の意地」があれば魂判定成功で1HPを残して気絶回避

### 死亡条件

PCの死亡は以下の条件で発生する。極めて重い結果なので、GMは事前に警告を出すことが推奨される。

| 条件 | 内容 |
|---|---|
| **HP0の状態で追加ダメージ** | 気絶中にさらにダメージを受けた場合、**死亡判定**を行う |
| **侵食率100%** | 怪異化してPCロスト。死亡ではないが、PLの手から離れる |
| **怪異のルールによる即死条件** | 一部の二級・一級怪異は「ルール違反3回で即死」などの条件を持つ |

### 死亡判定

HP0の状態でダメージを受けた場合、魂で判定する。

| 結果 | 内容 |
|---|---|
| スペシャル（6） | 奇跡の復活。HP1で復帰。全共鳴メーター+2 |
| 成功（4〜5） | まだ死んでいない。次の追加ダメージで自動死亡 |
| 失敗（2〜3） | 死亡。PLは死亡描写を行う権利を持つ |
| ファンブル（1） | 死亡。かつ怪異化の兆候（侵食率+30%の遺体が残る。シナリオ的な影響あり） |

<a id="h6"></a>
## H-6　ダウンタイムと長期回復

シナリオ間の休息期間。PLはダウンタイムに以下の行動から**2つ**を選べる。

| 行動 | 効果 |
|---|---|
| **休息** | HP全回復。負傷・重傷解除 |
| **訓練** | EP（経験値）+1 |
| **装備整備** | 破損装備の修復。メーカー修理の場合はダウンタイム消費 |
| **サイバネティクス施術** | 新しいサイバネティクスの追加（費用・EP消費あり） |
| **情報収集** | 次のシナリオに関するヒントを1つ入手 |
| **人脈構築** | NPCとの関係を1段階深める |
| **闇市場アクセス** | 非正規の装備・素材・情報を入手（リスクあり） |

## H-7　具体的な戦闘例

### 状況設定

- PC「アキラ」：体B（HP14）、武装型装備（武器修正+2）、ギフト「闘争本能」
- PC「ユウキ」：術B・視野C（HP10）、半装身型装備（武器修正+1）
- 護衛A：HP6 / 行動値2 / 特性：干渉（BLOCK）
- 護衛B：HP4 / 行動値3 / 特性：感染（INFECT）
- 核：討伐クロック8

### ラウンド1

**行動宣言フェイズ：**
- アキラ → 護衛Aに攻撃（ATTACK）
- ユウキ → 護衛Bに魔法攻撃（SPELL）

**解決（疾ランク順）：**

アキラの攻撃（体B = 3d6）：
```
出目：5, 3, 4
達成値：5（成功） → 共鳴ダイス：4（怒りメーター+1）
ダメージ = 2 + 武器修正2 + 闘争本能1（怒り3以上）= 5
護衛Aに5ダメージ → 護衛AのHP: 6 → 1
まだ耐久値を超えていないので連鎖ダメージなし
```

ユウキの魔法攻撃（術B = 3d6）：
```
出目：6, 2, 4
達成値：6（スペシャル！）→ 共鳴ダイス：4（浄化+1）
ダメージ = 3 + 武器修正1 + スペシャル追加1 = 5
護衛Bに5ダメージ → 護衛BのHP: 4 → 0
耐久値4以上のダメージ → 連鎖ダメージ発動！
連鎖：4ダメージが核に伝播（スペシャルなので×1.5 = 6ダメージ）
核のクロック：8 → 2
```

**護衛の行動：**
```
護衛A（HP1）：アキラに攻撃。行動値2。
アキラのHP: 14 → 12（2ダメージ）
護衛B：行動不能（このラウンド終了まで）
ただし感染（INFECT）特性が発動 → ユウキの恐怖メーター+1
```

### ラウンド2

アキラが護衛Aに追い打ち。護衛A残りHP1のため、成功すれば連鎖で核に残り6ダメージ中一部を叩き込める。

制限ラウンド内にクロック0到達で討伐成功。
