## プロジェクト概要

**IP名：** 怪異キル（kai-i-kill）
**ジャンル：** 近未来架空日本 × ホラー × サイバーパンク
**世界観要点：**
- 怪異（かいい）＝集合的な信念・噂が臨界を超えて物理的な形を取った「バグ」
- 怪異には「核（かく）」と「ルール」がある。ルールを知らずに戦うと死ぬ
- 魔法はインフラ（使いすぎると怪異を生む）
- 異能は内側から生まれる力（使うと自分が怪異に近づいていく）
- 三陣営：祓部（公的機関）・傭兵・無所属

## ★ 主文書：rules_unified.md（v3.0）の要点

`rules_unified.md` は `rules.md`（v1.0）と `rules_alt.md`（v2.1）を統合した最新版。
新しいセッションでは **まず `rules_unified.md` を読むこと**。

### ダイスシステム（rules_alt.mdベース）
ランク制ダイスプール：D=1d6 / C=2d6 / B=3d6 / A=4d6 / S=4d6+ボーナス
- 最大値を「達成値」として採用
- 4=部分成功（コストあり）/ 5=成功 / 6=完全成功（特殊効果）/ 1=ファンブル

### 属性7つ（統合版で新設計）
| 略 | 読み | 用途 |
|----|------|------|
| 体 | たい | 物理耐久・近接 |
| 疾 | はや | 先攻・回避 |
| 識 | しき | 調査・知識 |
| 判 | はん | 状況判断・解明 |
| 視野 | しや | 索敵・感知 |
| 術 | じゅつ | 魔法使用 |
| 魂 | こん | 異能・精神 |

### セッション三幕構造
1. **調査フェーズ**：解明鍵4つ（①正体 ②核の所在 ③被害パターン ④ルールの全容）を埋める
2. **解明完了宣言**：修正 −2〜+2（鍵数に応じて）→ 成功で討伐ウィンドウ獲得
3. **討伐フェーズ**：核護衛戦（守護者→核を順番に攻略）。討伐クロックを0にしたら勝利

### 《共鳴記録》（rules_alt.mdの感情システム）
6つの感情メーター（恐怖/怒り/哀愁/焦燥/渇望/浄化）。各0〜10。
行動の種類と文脈で上昇するメーターが決まる。10で臨界（コスト+恩恵が発生）。

### 侵食率
0〜100%。異能使用で上昇、絶対に下がらない。
段階：正常→変容の兆し（20%〜）→半化の影（50%〜）→臨界（80%〜）→怪異化（100%でキャラクター終了）

### 魔法システム（rules.mdベース、v3.0で強化）
8つの魔法言語（P / Igniscript / Lupis Surf / Ivyo / NGT / Monyx / P: / P'）
各言語に状況+1修正あり。魔法失敗時：GMが1d6、ランク別閾値超えで怪異誘発。

### 所属特典（rules.mdの具体的効果を継承）
| 所属 | 主な強み |
|------|---------|
| 祓部 | 識ランク+1段階（3回/セッション）・援軍要請1回 |
| 傭兵 | 装備1ランク上・コネクション+1段階（3回/セッション）・二つ名1つ（常時+1） |
| 無所属 | 視野+1（常時）・裏ルート1回・改造個体センサー1回 |

## 統合済みの改善内容（rules.md v1.0からの修正）

1. **怪異誘発判定の確率バグ** → 1d6ランク別閾値方式に変更（D=4以上で誘発50%、A以上=誘発なし）
2. **ダメージ解決の欠落** → 達成値+武器修正−防御値 の計算式を採用
3. **所属バランスの不均等** → 全所属に具体的な回数制限付き効果と常時効果を付与
4. **魔法言語の差別化** → 各言語に状況+1修正を追加（風味だけでなく機械的差異が生まれた）
5. **属性を6→7に拡張** → 疾（はや）を追加して先攻・回避を独立させた
