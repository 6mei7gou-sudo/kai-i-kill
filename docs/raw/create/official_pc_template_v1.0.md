## 本フォーマットについて

本書は運営が管理する「公式PC」の記入フォーマットだ。イベント・公式動画・Webサイト更新で使用するキャラクターを統一された書式で管理する。

**公式PCとNPCの違い：**

| 項目 | 公式PC（本フォーマット） | NPC（detail_*.md） |
|---|---|---|
| 対象 | 世界の住人としてPLに見せるキャラクター | GMのための道具 |
| 秘匿 | 基本的に公開情報（一部秘匿あり） | GM秘匿を含む |
| 更新 | イベントごとに状態が変化し更新される | 静的 |
| 用途 | イベント・動画での実プレイ、Web掲載 | シナリオ運用 |

**記入ルール：**
- 常体（だ・である調）で統一
- 記入欄は `【ここに記入】` で示す
- GM秘匿情報は `> **⚠ GM限定データ**` ブロック内に記載
- テーブルの空欄には `—` を入れる
